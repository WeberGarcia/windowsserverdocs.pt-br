# <a name="overview-of-the-dashboard-in-windows-server-essentials"></a>Visão geral do painel no Windows Server Essentials

>Aplica-se a: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials
 
 Windows Server Essentials e Windows Server 2012 R2 Standard com a função de experiência do Windows Server Essentials habilitada incluem um painel administrativo, que simplifica as tarefas executadas para gerenciar sua rede do Windows Server Essentials e o servidor. Usando o Windows Server Essentials Dashboard, você pode:  
  
-   Terminar de configurar seu servidor  
  
-   Acessar e executar tarefas administrativas comuns  
  
-   Exibir alertas de servidor e agir sobre eles  
  
-   Configurar e alterar as configurações do servidor  
  
-   Acesso ou pesquisar tópicos de ajuda na web  
  
-   Acessar recursos da comunidade na web  
  
-   Gerenciar contas de usuário  
  
-   Gerenciar dispositivos e backups  
  
-   Gerenciar o acesso e as configurações de discos rígidos e pastas de servidor  
  
-   Exibir e gerenciar aplicativos de suplementos  
  
-   Integre com os serviços online da Microsoft  
  
 Este tópico inclui:

##  <a name="BKMK_Design"></a>Recursos básicos do painel  
 O Windows Server Essentials Dashboard ajuda você a acessar rapidamente os principais recursos de gerenciamento e as informações do seu servidor. O painel inclui várias seções. A tabela a seguir descreve as seções.  
  
 
  
|Item|Recurso de painel|Descrição|  
|----------|-----------------------|-----------------|  
|1|Barra de navegação|Clique em uma seção na barra de navegação para acessar as informações e tarefas que são associadas essa seção. Cada vez que abrir o painel, o **Home** página é exibida por padrão.|  
|2|Guias de subseção|As guias subseção fornecem acesso a uma segunda camada de tarefas administrativas do Windows Server Essentials.|  
|3|Painel de lista|O modo de exibição de lista exibe os objetos que você pode gerenciar e inclui informações básicas sobre cada objeto.|  
|4|Painel de detalhes|O painel de detalhes exibe informações adicionais sobre um objeto que você selecionar no modo de exibição de lista.|  
|5|Painel de tarefas|O painel de tarefas contém links para ferramentas e informações que ajudarão a gerenciar as propriedades para um objeto específico (como uma conta de usuário ou um computador) ou para configurações globais para a categoria de objeto. No painel de tarefas é dividido nestas duas seções:<br /><br /> **Objeto tarefas** œ contém links para ferramentas e informações de ajudarão-lo a gerenciar as propriedades de um objeto que você selecionar no modo de exibição de lista (como uma conta de usuário ou um computador).<br /><br /> **Tarefas globais** œ contém links para ferramentas e informações de ajudarão-lo a gerenciar tarefas globais para uma área de recurso. Tarefas globais incluem tarefas para adicionar novos objetos, defina a política e assim por diante.|  
|6|Informações e configurações|Esta seção fornece acesso direto para as configurações do servidor e um link de ajuda para obter informações sobre a página do painel que você está visualizando.|  
|7|Alertas de status|O status de alertas fornece uma indicação visual rápida sobre a integridade do servidor. Clique na imagem de alerta para exibir alertas críticas e importantes.<br /><br /> **Observação:** no Windows Server Essentials e no Windows Server 2012 R2 Standard com a função de experiência do Windows Server Essentials instalada, o status de alertas está disponível no **informações e configurações** guia.|  
|8|Barra de status|A barra de status exibe o número de objetos que aparecem na exibição de lista. Add-in aplicativos também podem exibir outros status.|  
  
##  <a name="BKMK_Home"></a>Recursos da página inicial do painel  
 Quando você abre o painel, o **Home** página é exibida por padrão com o **instalação** categoria exibida. O **Home** página do Windows Server Essentials painel fornece acesso rápido a tarefas e informações que ajudam a personalizar seu servidor e configure principais recursos. A Home page é composto de quatro áreas funcionais, que expõem tarefas de configuração e informações para as opções que você selecionar. A tabela a seguir descreve os recursos.  
  
|Item|Recurso|Descrição|  
|----------|-------------|-----------------|  
|1|Barra de navegação|Clique em uma seção na barra de navegação para acessar as informações e tarefas que são associadas essa seção. Cada vez que abrir o painel, o **Home** página é exibida por padrão.|  
|2|Painel de categoria|Esse painel exibe as áreas de recursos que oferecem acesso rápido a informações e ferramentas de configuração que ajudam você a configurar e personalizar o servidor. Clique em uma categoria para exibir as tarefas e recursos que estão associados essa categoria.|  
|3|Painel de tarefas|Esse painel exibe links de tarefas e informações que se aplicam a uma categoria selecionada. Clique em uma tarefa ou recurso para exibir informações adicionais.|  
|4|Painel de ações|Esse painel fornece uma breve descrição de um recurso ou uma tarefa e fornece links que abrem assistentes de configuração e páginas de informações. Clique no link para fazer mais nada.|  
  
##  <a name="BKMK_Features"></a>Seções administrativas do painel  
 O Windows Server Essentials Dashboard organiza as informações de rede e tarefas administrativas em áreas funcionais. A página de gerenciamento para cada área funcional fornece informações sobre os objetos associados nessa área, por exemplo **usuários**, ou **dispositivos**. A página de gerenciamento também inclui tarefas que você pode usar para exibir ou alterar configurações ou para executar programas que automatizam tarefas que exigem várias etapas.  
  
 A tabela a seguir descreve as seções administrativas do painel que estão disponíveis por padrão após a instalação. A tabela também lista as tarefas que estão disponíveis em cada seção.  
  
|Seção|Descrição|  
|-------------|-----------------|  
|Casa|O **Home** página é exibida por padrão cada vez que abrir o painel. Ele inclui tarefas e informações nas seguintes categorias:<br /><br /> **INSTALAÇÃO** œ concluir as tarefas nessa categoria para configurar o servidor pela primeira vez. Para obter informações sobre essas tarefas, consulte [instalar e configurar o Windows Server Essentials](../install/Install-and-Configure-Windows-Server-Essentials.md).<br /><br /> **EMAIL** œ escolha uma opção nessa categoria integrar um serviço de email com o servidor.<br /><br /> **Observação:** nessa categoria só está disponível no Windows Server Essentials.<br /><br /> **SERVIÇOS** œ escolher uma tarefa nessa categoria para integrar os serviços online da Microsoft com o servidor.<br /><br /> **Observação:** nessa categoria só está disponível no Windows Server Essentials e no Windows Server 2012 R2 Standard com a função de experiência do Windows Server Essentials habilitada.<br /><br /> **ADD-INS** œ clicar nessa categoria para instalar suplementos valiosos para sua empresa.<br /><br /> **STATUS rápido** œ exibe o status do servidor de alto nível. Clique em um status para ver as opções de configuração e informações para esse recurso. Se você concluir todas as tarefas na categoria de instalação, essa categoria aparece na parte superior do painel de categoria.<br /><br /> **AJUDAR** œ usar a caixa de pesquisa para procurar ajuda na Web. Clique no link para visitar o site para a opção de suporte selecionado.|  
|Usuários|Para os usuários acessem os recursos fornecidos pelo Windows Server Essentials, você precisa criar contas de usuário usando o painel do Windows Server Essentials. Depois de criar contas de usuário, você pode gerenciar as contas, usando as tarefas que estão disponíveis na **usuários** página do painel. Tarefas que você pode executar nesta página incluem:<br /><br /> -Ver uma lista de contas de usuário.<br /><br /> -Exibir e gerenciar as propriedades de conta de usuário.<br /><br /> -Ativar ou desativar contas de usuário.<br /><br /> -Adicionar ou remover contas de usuário.<br /><br /> -Atribua contas de rede local a contas de serviços online da Microsoft se o servidor está integrado com o Office 365.<br /><br /> -Alterar senhas de conta de usuário e gerenciar a política de senha.<br /><br /> Para obter informações sobre como gerenciar contas de usuário, consulte [gerenciar contas de usuário](Manage-User-Accounts-in-Windows-Server-Essentials.md).|  
|Grupos de usuários|**Observação:** esse recurso está disponível apenas no Windows Server Essentials e no Windows Server 2012 R2 Standard, com a função de experiência do Windows Server Essentials habilitada.<br /><br /> Tarefas que você pode executar nesta página incluem:<br /><br /> -Ver uma lista de grupos de usuários.<br /><br /> -Exibir e gerenciar grupos de usuários.<br /><br /> -Adicionar ou remover grupos de usuários.|  
|Grupos de distribuição|**Observação:** esse recurso está disponível apenas no Windows Server Essentials e no Windows Server 2012 R2 Standard, com a função de experiência do Windows Server Essentials habilitada. ##  <a name="BKMK_AccessDb"></a>Acessar o painel do Windows Server Essentials  
 Você pode acessar o painel por meio de um dos métodos a seguir. O método que você escolher depende se você estiver acessando o painel do servidor, de um computador que esteja conectado à rede Windows Server Essentials ou de um computador remoto.  
  
 Para ajudar a manter uma rede segura, somente os usuários com permissões administrativas podem acessar o painel do Windows Server Essentials. Além disso, você não pode usar a conta de administrador interno para fazer logon no Windows Server Essentials painel da barra inicial.  
  
###  <a name="BKMK_Server"></a>Acessar o painel do servidor  
 Quando você instala o Windows Server Essentials, o processo de instalação cria um atalho para o painel no **iniciar** tela e também na área de trabalho. Se o atalho estiver ausente esses locais, você pode usar o **pesquisa** painel para encontrar e executar o programa de painel.  
  
##### <a name="to-access-the-dashboard-from-the-server"></a>Para acessar o painel do servidor  
  
-   Fazer logon servidor como administrador e siga um destes procedimentos:  
  
    -   Sobre o **iniciar** de tela, clique em **painel**.  
  
    -   Na área de trabalho, clique duas vezes em **painel**.  
  
    -   No **pesquisa** painel, digite **painel**e clique em **painel** nos resultados da pesquisa.  
  
###  <a name="BKMK_Network"></a>Acessar o painel de um computador que esteja conectado à rede  
 No Windows Server Essentials, depois de ingressar um computador à rede, os administradores podem usar a barra inicial para acessar o painel de servidor do computador.  
  
> [!WARNING]
>  No Windows Server Essentials, para conectar-se para o painel do computador cliente, clique com botão direito no ícone e, em seguida, abra o painel no menu de contexto.  
  
##### <a name="to-access-the-dashboard-by-using-the-launchpad"></a>Para acessar o painel por meio da barra inicial  
  
1.  Em um computador que tenha ingressado em rede, abra o **Launchpad**.  
  
2.  No menu na barra inicial, clique em **painel**.  
  
3.  No painel **login** página, digite suas credenciais de administrador de rede e pressione ENTER.  
  
     Abre uma conexão remota para o painel do Windows Server Essentials.  
  
###  <a name="BKMK_Remote"></a>Acessar o painel de um computador remoto  
 Quando você estiver trabalhando em um computador remoto, você pode acessar o painel do Windows Server Essentials usando o site de acesso via Web remoto.  
  
##### <a name="to-access-the-dashboard-by-using-remote-web-access"></a>Para acessar o painel usando o acesso via Web remoto  
  
1.  No computador remoto, abra um navegador da Internet, como o Internet Explorer.  
  
2.  Na barra de endereços do navegador de Internet, digite o seguinte:**https://<DomainName\>/remote**, onde *DomainName* é o nome de domínio externo da sua organização (como contoso.com).  
  
3.  Digite seu nome de usuário e senha para efetuar login no site do acesso via Web remoto.  
  
4.  No **computadores** seção o acesso via Web remoto**Home** página; Localize seu servidor e clique em **conectar**.  
  
     Abre uma conexão remota para o painel.  
  
    > [!NOTE]
    >  Se o servidor não aparecem no **computadores** seção da Home page, clique em **conectar a computadores mais**, localize o servidor na lista e, em seguida, clique no servidor para se conectar a ela.  
    >   
    >  Para dar uma permissão de usuário para acessar o painel de acesso via Web remoto, abra a página de propriedades da conta de usuário e, em seguida, selecione o **Server painel** opção no **acesso em qualquer local** guia.  
  
##  <a name="BKMK_UseSafeMode"></a>Use o modo de segurança  
 O recurso de modo de segurança do Windows Server Essentials monitora os suplementos que estão instalados no servidor. Se o painel ficar instável ou parar de responder, o modo de segurança detecta e isola suplementos que podem estar causando o problema e os exibe no **configurações do modo de segurança** na próxima vez que abrir o painel de página. Do **configurações do modo de segurança** página, você pode desabilitar e habilitar suplementos um um para determinar qual suplemento está causando o problema. Você pode optar por deixar o suplemento desabilitado para inicializações futuras, ou você pode desinstalar o suplemento.  
  
 Você pode exibir uma lista de todos os complementos que estão instalados no servidor a qualquer momento. Você também pode abrir o painel no modo de segurança, o que desabilita automaticamente todos os suplementos não são da Microsoft. Windows Server Essentials também permite que você acessar o painel no modo de segurança de outro computador na rede.  
  
#### <a name="to-view-a-list-of-installed-add-ins"></a>Para exibir uma lista de instalar suplementos  
  
-   No painel, clique em **ajudar**e clique em **configurações do modo de segurança**.  
  
#### <a name="to-open-the-dashboard-in-safe-mode"></a>Para abrir o painel no modo de segurança  
  
-   No **pesquisa** painel, digite **seguro**e clique em **(modo de segurança) do painel** nos resultados da pesquisa.  
  
#### <a name="to-open-the-dashboard-in-safe-mode-from-another-computer-on-the-network"></a>Para abrir o painel no modo de segurança de outro computador na rede  
  
1.  Em um computador que esteja conectado à rede, abra o Windows Server Essentials Launchpad e, em seguida, clique em **painel**.  
  
2.  Na página painel entrar, digite o nome de usuário e senha para uma conta que tenha permissão para fazer logon no servidor, selecione o **permitir que eu selecionar quais suplementos para carregar** caixa de seleção e, em seguida, clique na seta para fazer logon.
