# <a name="step-8-run-the-windows-server-essentials-best-practices-analyzer"></a>Etapa 8: Executar o analisador de práticas recomendadas do Windows Server Essentials

>Aplica-se a: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Quando terminar de migrar suas configurações e dados ao Windows Server Essentials, você deve executar o Windows Server Essentials BPA Best Practices Analyzer () no servidor de destino. A ferramenta BPA examina um servidor que esteja executando o Windows Server Essentials e apresenta um relatório que descreve problemas e fornece recomendações para resolvê-los. As recomendações são desenvolvidas pela organização de suporte de produto do Windows Server Essentials.  
  
### <a name="to-run-the--windows-server-essentials-bpa-on-the-destination-server"></a>Para executar o Windows Server Essentials BPA no servidor de destino  
  
1.  Faça logon no servidor de destino como administrador e, em seguida, abra o Gerenciador do servidor.  
  
2.  No Gerenciador do servidor, clique no **a experiência do Windows Server Essentials** guia.  
  
3.  No painel de detalhes, selecione **analisador de práticas recomendadas**.  
  
4.  Clique em **tarefas**e clique em **varredura BPA iniciar**.  
  
5.  Examine cada mensagem BPA e siga as instruções para resolver problemas, se necessário.  
  
## <a name="next-steps"></a>Próximas etapas  
 Você executou o analisador de práticas recomendadas do Windows Server Essentials. A migração para o Windows Server Essentials agora está concluída.
