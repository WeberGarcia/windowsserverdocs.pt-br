# <a name="release-documentation-for-windows-server-essentials"></a>Documentação de versão para Windows Server Essentials

>Aplica-se a: Windows Server 2016 Essentials

Bem-vindo à documentação de versão para o software de servidor do Windows Server 2016 Essentials. Este tópico contém informações sobre problemas que você pode encontrar ao instalar e executar o software de servidor Windows 2016 Server Essentials. Você deve examinar este tópico antes de instalar e executar o Windows Server 2016 Essentials na sua rede.  
  
## <a name="client-computers"></a>Computadores cliente  
  
### <a name="install-all-available-critical-and-security-updates-from-windows-update"></a>Instale todas as atualizações Críticas e de Segurança do Windows Update  

Antes de conectar um computador cliente ao servidor, certifique-se de que você abre o Windows Update no computador cliente e instale todas as atualizações críticas e de segurança disponíveis.  
  
### <a name="client-computer-restore-may-not-succeed-after-migration-to-windows-10"></a>Restauração do computador cliente pode não ter êxito após a migração para o Windows 10  
 Depois de atualizar um computador de cliente do Windows 8.1 para Windows 10, a restauração do computador cliente pode não funcionar corretamente. Você precisará reconectar-se o cliente para o Windows Server 2016 Essentials.
