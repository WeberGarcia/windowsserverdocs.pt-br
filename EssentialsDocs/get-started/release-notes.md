# <a name="release-documentation-for-windows-server-essentials"></a>Documentação da versão do Windows Server Essentials

>Aplica-se a: Windows Server 2016 Essentials

Bem-vindo à documentação da versão do software de servidor Windows Server 2016 Essentials. Este tópico contém informações sobre problemas que podem ocorrer quando você instala e executar o software de servidor de 2016 do Windows Server Essentials. Você deve revisar neste tópico antes de instalar e executar o Windows Server 2016 Essentials em sua rede.  
  
## <a name="client-computers"></a>Computadores cliente  
  
### <a name="install-all-available-critical-and-security-updates-from-windows-update"></a>Instale todas as atualizações críticas e de segurança disponíveis do Windows Update  

Antes de conectar um computador cliente para o servidor, certifique-se de que você abre o Windows Update no computador cliente e instale todas as atualizações críticas e de segurança disponíveis.  
  
### <a name="client-computer-restore-may-not-succeed-after-migration-to-windows-10"></a>Restauração do computador cliente não pode ter sucesso após a migração para o Windows 10  
 Depois de atualizar um computador cliente do Windows 8.1 para o Windows 10, restauração do computador cliente talvez não funcionem corretamente. Você precisará reconectar o cliente para o Windows Server 2016 Essentials.
