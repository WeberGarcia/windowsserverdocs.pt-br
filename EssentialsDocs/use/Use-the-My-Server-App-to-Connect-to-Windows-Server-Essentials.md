# <a name="use-the-my-server-app-to-connect-to-windows-server-essentials"></a>Use o aplicativo meu servidor para se conectar ao Windows Server Essentials

>Aplica-se a: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Os aplicativos de meu servidor para Windows Server Essentials permitem que você se conectar aos recursos e realizar tarefas administrativas claras em seu servidor Windows Server Essentials do seu notebook, computador ou dispositivo Surface. No meu servidor, você pode gerenciar usuários, dispositivos e alertas e trabalhar com arquivos compartilhados no servidor. Quando você estiver offline, você pode continuar a trabalhar com arquivos acessados recentemente em meu servidor; as alterações offline são automaticamente sincronizadas com o servidor na próxima vez que você se conectar.  
  
 Se o servidor está executando o sistema operacional Windows Server Essentials, use o aplicativo meu servidor original. Se o servidor está executando o sistema operacional Windows Server Essentials, você deve usar o aplicativo atualizado do meu Server 2012 R2. Ambos os aplicativos podem ser baixados no [aplicativos para Windows](https://windows.microsoft.com/windows-8/apps).  
  
> [!TIP]

>  Para acessar recursos no Windows Server Essentials do seu Windows Phone, use o aplicativo de telefone de meu servidor (para Windows Server Essentials) ou o aplicativo de telefone de meu Server 2012 R2 (para Windows Server Essentials). Para obter informações sobre o aplicativo de telefone de meu servidor, consulte [usar o aplicativo meu servidor para Windows Phone](Work-Remotely-in-Windows-Server-Essentials.md#BKMK_2). Para obter informações sobre o aplicativo de telefone de meu Server 2012 R2, veja a entrada de blog [aplicativos meu Server 2012 R2 Windows e Windows Phone](http://blogs.technet.com/b/sbs/archive/2013/11/19/my-server-2012-r2-windows-and-windows-phone-apps.aspx).

##  <a name="BKMK_WhatsNew"></a>O que é novo no meu Server 2012 R2  
 Além dos recursos disponíveis no meu servidor, meu Server 2012 R2 oferece os seguintes novos recursos para os clientes com o Windows Server Essentials:  
  
-   **Acesso a sites de equipe do SharePoint Online e bibliotecas** -aproveitar Microsoft Office 365 integração com o Windows Server Essentials para acessar sites de equipe do SharePoint Online e abrir documentos em suas bibliotecas do SharePoint Online no meu Server 2012 R2.  
  
-   **Conexões de área de trabalho remota para os computadores cliente e servidor** -uso **conexão remota** no meu Server 2012 R2 para se conectar ao seu servidor Windows Server Essentials e computadores cliente usando a área de trabalho remota.  
  
##  <a name="BKMK_OS"></a>Requisitos de sistema operacional  
 Para usar meu servidor ou meu Server 2012 R2 em seu notebook, computador ou dispositivo Surface, o computador deve ter um dos seguintes sistemas operacionais: Windows 8.1, Windows RT 8.1, Windows 8 ou Windows RT.  
  
##  <a name="BKMK_Install"></a>Instalar meu servidor  
 Você pode baixar o aplicativo meu servidor apropriado a partir do [aplicativos para Windows](https://windows.microsoft.com/windows-8/apps) armazenar. Se o computador tiver o sistema operacional Windows 8 ou Windows RT, e você deseja se conectar ao servidor na intranet usando o nome do servidor, você também precisa instalar um certificado do servidor.  
  
#### <a name="to-install-my-server-or-my-server-2012-r2-on-your-windows-based-laptop-pc-or-surface-device"></a>Para instalar meu servidor ou meu Server 2012 R2 em seu notebook baseado no Windows, computador ou dispositivo Surface  
  
1.  Instalar o aplicativo meu servidor apropriado de [aplicativos para Windows](https://windows.microsoft.com/windows-8/apps) em seu notebook baseado no Windows, computador ou dispositivo Surface.  
  
    |Sistema operacional de servidor|Baixar|  
    |-----------------------------|--------------|  
    | Windows Server Essentials|[Meu servidor](https://apps.microsoft.com/webpdp/app/19d94f81-db21-4234-8b49-806694dbbaea)|  
    | Windows Server Essentials|[Meu Server 2012 R2](https://apps.microsoft.com/webpdp/app/67e86695-bda3-4f32-96c4-2e20e56f1cf3)|  
  
2.  Se você deseja ser capaz de usar o nome do servidor para se conectar ao servidor Windows Server Essentials por uma intranet, e seu dispositivo tem o sistema operacional Windows 8 ou Windows RT, instale o certificado de CAROOT.cer padrão no seu computador, executando o procedimento a seguir. Instalação manual de um certificado não é necessário no Windows 8.1 e Windows RT 8.1.  
  
###  <a name="BKMK_InstallCert"></a>Para instalar um certificado para meu servidor em seu dispositivo Windows 8, Windows 8.1 ou Windows RT  
  
1.  Abra um navegador da web em seu computador e baixe o certificado de padrão, CAROOT.cer, do servidor. Para fazer isso, digite o seguinte, substituindo o nome do seu servidor (por exemplo, **marketing.contoso.com**) para <*nomedoservidor*>:  
  
     **http://<servername\>/Connect/default.aspx?Get=caroot.cer**  
  
2.  Instale o certificado:  
  
    1.  Sobre o **iniciar** página, abrir **Explorador de arquivos**.  
  
    2.  Certifique-se de que os itens ocultos e extensões de nome de arquivo são exibidas: no **exibição** guia, o **Mostrar/ocultar** grupo, selecione o **itens ocultos** e **extensões de nome de arquivo** caixas de seleção.  
  
    3.  Navegue até a pasta que contém o arquivo CAROOT.cer que você acabou de baixar.  
  
    4.  Clique com botão direito do arquivo CAROOT.cer e clique em **abrir**.  
  
    5.  No **certificado** caixa de diálogo, clique em **instalar o certificado**.  
  
    6.  Selecione **Máquina Local** como o local de instalação e clique em **próxima**.  
  
    7.  No **repositório de certificados** página do assistente, selecione **colocar todos os certificados no repositório a seguir**e use **procurar** para escolher o **autoridades de certificação confiáveis** armazenar. Clique em **concluir**.  
  
##  <a name="BKMK_UseServer"></a>Usar meu servidor  
 Para começar a usar o aplicativo meu servidor ou meu Server 2012 R2, abra o aplicativo e faça um tour rápido de seus recursos.  
  
#### <a name="to-open-my-server-or-my-server-2012-r2"></a>Para abrir meu servidor ou meu Server 2012 R2  
  
1.  Abrir meu servidor (para Windows Server Essentials) ou meu Server 2012 R2 (para Windows Server Essentials) a partir do **iniciar** página do seu dispositivo Windows.  
  
2.  Entrar no servidor Windows Server Essentials usando sua conta no servidor. Para identificar o servidor:  
  
    -   Em um ambiente de fora do local, use o nome de domínio do servidor s por exemplo, **marketing.contoso.com**.  
  
    -   Se você estiver conectando local por uma intranet, use o nome do servidor no exemplo anterior, o nome do servidor é **marketing**. (Se você estiver usando um computador Windows 8 ou Windows RT para se conectar no local e você deseja usar esse método, você deve instalar um certificado de CAROOT.cer do servidor. Para obter mais informações, consulte [para instalar um certificado para meu servidor em seu dispositivo Windows 8, Windows 8.1 ou Windows RT](#BKMK_InstallCert).)

##  <a name="BKMK_ConnectServer"></a>Como conectar-se ao seu servidor de rede local  
 Para usar com êxito o aplicativo meu servidor no Windows Server Essentials para Windows Phone, Windows 8 e Windows 8.1, primeiro você deve instalar o certificado do servidor em seu dispositivo. Isso conecta o dispositivo ao servidor que executa o Windows Server Essentials em sua rede local. Use os procedimentos a seguir para se conectar ao servidor na rede local e instalar o certificado do servidor no seu Windows Phone ou em um computador que executa o Windows 8 ou Windows 8.1.  
  
#### <a name="to-connect-to-your-server-from-your-windows-phone"></a>Para se conectar ao servidor do seu Windows Phone  
  
1.  No seu Windows Phone que executa o Windows 8 ou Windows 8.1, abra o Internet Explorer.  
  
2.  Na barra de endereços, digite **http://<yourservername\>/connect/default.aspx?get=caroot.cer**, e pressione Enter.  
  
3.  Quando você for solicitado a instalar o certificado caroot.cer, clique em **instalar**.  
  
4.  Aguarde a conclusão da instalação do certificado.  
  
    > [!NOTE]
    >  Após a instalação do certificado for concluída, você pode entrar aplicativo meu servidor para Windows Phone usando suas credenciais de rede e o nome do seu servidor.  
  
#### <a name="to-connect-to-your-server-in-your-local-network-from-a-computer-running-windows-8-or-windows-81"></a>Para se conectar ao servidor na rede local de um computador executando o Windows 8 ou Windows 8.1  
  
1.  Em um computador que executa o Windows 8 ou Windows 8.1, abra o Internet Explorer.  
  
2.  Na barra de endereços, digite **http://<yourservername\>/connect/default.aspx?get=caroot.cer**, e pressione Enter.  
  
3.  Quando você for solicitado a instalar o certificado caroot.cer, clique em **abrir**.  
  
4.  No **certificado** caixa de diálogo, clique em **Instalar certificado**.  
  
5.  No Assistente de importação do certificado, selecione **Máquina Local** como o local do repositório.  
  
6.  Selecione **colocar todos os certificados no repositório a seguir**e, em seguida, navegue até o **autoridades de certificação confiáveis** local.  
  
7.  Siga as instruções para concluir o assistente.  
  
    > [!NOTE]
    >  Após a instalação do certificado for concluída, você pode entrar aplicativo meu servidor para Windows 8 ou Windows 8.1 usando suas credenciais de rede e o nome do seu servidor.
