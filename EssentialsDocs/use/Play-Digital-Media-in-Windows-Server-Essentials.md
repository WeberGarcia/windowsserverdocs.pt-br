##  <a name="BKMK_1"></a>Visão geral de mídia digital  
 Mídia digital se refere ao áudio, vídeo e conteúdo de foto que tenha sido codificado (digitalmente compactados). Conteúdo de codificação envolve a conversão de entrada de áudio e vídeo em um arquivo de mídia digital, como um arquivo de mídia do Windows. Depois de mídia digital é codificada, ele pode ser facilmente manipulado, distribuído e executado pelos computadores e facilmente são transmitidos por redes de computador.  
  
 Exemplos dos tipos de mídia digital incluem: Windows Media Audio (WMA), Windows Media Video (WMV), MP3, JPEG e AVI. Para saber mais sobre os tipos de mídia digital são suportados pelo Windows Media Player, veja [suportados pelo Windows Media Player de tipos de arquivo](https://support.microsoft.com/kb/316992).  
  
### <a name="why-would-i-want-to-stream-my-digital-media"></a>Por que eu quiser transmitir mídia digital?  
 Muitas pessoas armazenam fotos, vídeo e música em pastas compartilhadas no Windows Server Essentials. Pode haver momentos em que você queira fazer a seguir:  
  
-   **Assista a vídeos**. O servidor pode ser usado para armazenar e transmitir grandes coleções de vídeos e programas de TV gravados mostra seus computadores ou reprodução de outra dispositivos em sua rede. Você pode transmitir vídeos para um Xbox 360 ou para um computador usando o Windows Media Player.  
  
-   **Reproduzir música**. Quando você ativar o compartilhamento de mídia para o **música** pasta compartilhada, você pode acessar suas músicas de dispositivos que dão suporte a conexão de mídia do Windows. Você não precisa ativar ou configurar as contas de usuário para transmitir do **música** pasta compartilhada depois que o compartilhamento está ativado.  
  
-   **Apresentar apresentações de slides de fotos**. Você pode armazenar suas fotos digitais no **fotos** pasta no servidor compartilhada e, em seguida, acessá-los de qualquer computador ou de um Xbox 360 estiver conectado a uma TV em casa ou no escritório. Você pode assistir a apresentações de slides de fotos, que é como transformar seus programas de TV em um quadro de imagem grande.  
  
### <a name="sharing-copy-protected-media"></a>Compartilhamento de mídia protegidos contra cópia  
  Windows Server Essentials não oferece suporte a compartilhamento de mídia protegidos contra cópia. Isso inclui a música que está adquirida por meio de uma loja de música online.  
  
 Mídia Copy-protected pode ser reproduzida apenas no computador ou dispositivo que você usou para comprá-lo. Proteção contra cópia impede a reprodução de mídia em mais de um computador ou dispositivo, mesmo se você copiar a mídia ao seu servidor e reproduzi-lo a partir daí. No entanto, você pode armazenar a mídia protegidos contra cópia no Windows Server Essentials e continuar a reproduzir a mídia no computador ou dispositivo que você usou para comprá-lo.

####  <a name="BKMK_WMC"></a>Procure e reproduzir arquivos de mídia de um computador que esteja executando o Windows Media Center  
  
1.  Clique em **iniciar**, clique em **todos os programas**e clique em **Windows Media Center**.  
  
2.  Sobre o **Windows Media Center** página, role até o tipo de mídia que você está procurando e clique em biblioteca de mídia.  
  
3.  Procurar manualmente o arquivo que você está interessado, ou clique em **pesquisa** e digite o nome do arquivo que deseja localizar.  
  
4.  Clique na imagem de arquivo de mídia para exibir ou executá-lo.  
  
####  <a name="BKMK_MWP"></a>Procure e reproduzir arquivos de mídia de um computador que está executando o Windows usando o Windows Media Player  
  
-   O computador ou dispositivo de mídia, abra **Windows Media Player** e procure por sua biblioteca de mídia.  
  
    > [!NOTE]
    >  As etapas de pesquisa variam dependendo da versão do Windows Media Player que você está usando. Para obter informações detalhadas, consulte a Ajuda para a sua versão.  
  
####  <a name="BKMK_Xbox"></a>Procure e reproduzir arquivos de mídia usando o Xbox 360  
  
1.  Conecte seu console Xbox 360 à sua rede doméstica usando uma conexão com ou sem fio.  
  
2.  No computador que esteja executando o Windows Server Essentials, ative o compartilhamento de mídia. Para obter mais informações, consulte o tópico [ativar ou desativar de streaming de mídia](../manage/Manage-Digital-Media-in-Windows-Server-Essentials.md#BKMK_4).  
  
3.  Para reproduzir arquivos de mídia digital usando o console do Xbox 360:  
  
    1.  Vá para **My Xbox**e, em seguida, selecione **biblioteca de vídeo**, **biblioteca de músicas**, ou **biblioteca de imagens**, dependendo do tipo de mídia que deseja exibir ou reproduzir.  
  
    2.  Selecione o nome do seu servidor.  
  
        > [!NOTE]
        >  Se o nome do seu servidor não estiver listado, selecione **computador**e clique em **Conexão de teste**.  
  
    3.  Navegue pela lista de arquivo e selecione o item que você deseja reproduzir.  
  
####  <a name="BKMK_Other"></a>Procure e reproduzir arquivos de mídia usando outros players de mídia digital ou receptores que são compatíveis com o Windows Server Essentials  
  
1.  Vá para o [Centro de compatibilidade do Windows](https://www.microsoft.com/windows/compatibility/CompatCenter/Home) e verifique se o player de mídia digital ou receptor aparece na lista de dispositivos compatíveis.  
  
2.  Porque a pesquisa etapas variam de acordo com o player de mídia digital que você está usando, consulte a Ajuda para o seu dispositivo para obter instruções detalhadas.  
  
####  <a name="BKMK_SharedFolders"></a>Procure e reproduzir arquivos de mídia usando o recurso de pastas compartilhadas da barra inicial  
  
1.  Login para o Windows Server Essentials Launchpad.  
  
2.  Na barra inicial, clique em **pastas compartilhadas**. Uma janela de Windows Explorer abre e exibe as pastas compartilhadas no servidor.  
  
3.  No **pesquisa**, digite o nome de um arquivo de mídia. Os resultados da sua consulta de pesquisa são exibidos.  
  
    > [!NOTE]
    >  Como opção, você pode também clique duas vezes em uma pasta compartilhada para procurar o conteúdo da pasta.  
  
####  <a name="BKMK_RWA2"></a>Procure e reproduzir mídia compartilhada usando o acesso via Web remoto  
  
1.  Fazer logon em acesso via Web remoto.  
  
2.  Clique em **pastas compartilhadas**. O **pastas compartilhadas** seção da página da web exibe uma lista de pastas compartilhadas no servidor.  
  
3.  Clique duas vezes em uma pasta para exibir o conteúdo da pasta.  
  
###  <a name="BKMK_SendToDevice"></a>Enviar arquivos de mídia no Windows Server Essentials para o Windows Media Player, Xbox 360, ou para um player de mídia digital em rede na rede  
 Use **Windows Media Player** para procurar o arquivo de mídia que você deseja. Clique com botão direito no arquivo de mídia e, em seguida, clique em **reproduzir em** para enviar o arquivo de mídia para um dispositivo de mídia em rede.  
  
##  <a name="BKMK_3"></a>Reproduzir arquivos de mídia digital compartilhado de um local remoto  
 Você pode reproduzir arquivos de mídia quando você estiver longe de sua rede do Windows Server Essentials usando o acesso via Web remoto. Você pode usar um telefone celular, um computador remoto ou um player de mídia digital para procurar e reproduzir arquivos de mídia compartilhada armazenados no seu servidor.  
  
#### <a name="to-play-shared-media-files-when-you-are-away-from-the-network"></a>Para reproduzir arquivos de mídia compartilhada quando você estiver longe da rede  
  
1.  Abra um navegador da Internet.  
  
2.  Acesse o site de acesso via Web remoto. Tipo **https://<YourDomainName\>/remote** na barra de endereços do navegador da Internet e pressione Enter.  
  
    > [!NOTE]
    >  *< YourDomainName\ >* é um espaço reservado. Ele será um nome exclusivo ao seu servidor para o endereço que você digita parecerá **https://contoso.com/remote**. Se você não souber o nome do seu domínio, peça ao administrador que escolheu o nome de domínio quando a funcionalidade de acesso remoto foi definida no servidor. Para obter mais informações, consulte [ativar o acesso via Web remoto](../manage/Manage-Remote-Web-Access-in-Windows-Server-Essentials.md#BKMK_TurnOnRWA).  
  
3.  Na página acesso via Web remoto entrar, digite seu nome de conta de usuário e senha e, em seguida, clique na seta.  
  
4.  Use qualquer método que você deseja procurar o arquivo de mídia que você deseja reproduzir.  
  
    > [!NOTE]

    >  Para obter informações sobre os vários métodos de pesquisa, consulte [procurar e reproduzir arquivos de mídia no Windows Server Essentials de um computador ou o player de mídia digital na rede](Play-Digital-Media-in-Windows-Server-Essentials.md#BKMK_2.1).  

    >  Para obter informações sobre os vários métodos de pesquisa, consulte [procurar e reproduzir arquivos de mídia no Windows Server Essentials de um computador ou o player de mídia digital na rede](../use/Play-Digital-Media-in-Windows-Server-Essentials.md#BKMK_2.1).  

  
5.  Quando o nome do arquivo de mídia for exibida, clique no nome de arquivo para reproduzir a mídia.

##  <a name="BKMK_5"></a>Opções de formato de download  
 Há duas opções para baixar arquivos. Essas opções estão disponíveis apenas quando você estiver baixando vários arquivos ou uma pasta para um computador baseado no Windows.  
  
 Escolha a opção a seguir que atenda às suas necessidades para downloads:  
  
-   **Compactado ZIP arquivo (.zip)**  
  
     Ao compactar um arquivo cria uma versão compactada do arquivo que é menor do que o arquivo original. A versão compactada do arquivo tem uma extensão de nome de arquivo. zip. Tipos de arquivo que são reduziu a maior compactação são tipos de arquivo orientados a texto (como. txt,. doc e. xls) e arquivos de elementos gráficos que tipos de arquivo não compactado de uso (por exemplo,. bmp). Alguns arquivos gráficos, como arquivos. jpg e. gif, já usam compactação e o tamanho do arquivo é reduzido pouquíssima por compactados. Além disso, um documento do Word que contém uma série de elementos gráficos não é reduzido quanto um documento que é basicamente texto.  
  
    > [!NOTE]
    >  Essa opção fornece suporte limitado para nomes de arquivo internacionais.  
  
-   **Arquivos executáveis autoextraíveis (.exe)**  
  
     Um arquivo executável autoextraíveis é um arquivo que você pode baixar que combina o programa de descompactação (executável) com os arquivos compactados. Quando você executa o programa executável, ele descompacta automaticamente os arquivos compactados. Isso é uma maneira comum de distribuir dados compactados sem se preocupar se o destinatário tem o utilitário de descompactação certo.  
  
    > [!NOTE]
    >  Essa opção dá suporte a caracteres Unicode.  
  
 Antes de começa o download real, o arquivo .exe ou. zip é criado. Dependendo do número de arquivos e o tamanho total dos arquivos sejam baixados, isso pode levar vários minutos. Depois que o download de arquivo é criado, baixando o arquivo ocorre em segundo plano. Isso permite que você continue trabalhando enquanto o processo de download for concluído.  
  
##  <a name="BKMK_6"></a>Ferramenta fácil de carregamento de arquivo  
 A ferramenta de carregamento de arquivo fácil simplifica o processo de carregamento de arquivos no seu servidor Windows Server Essentials. Você pode adicionar quantos arquivos que você deseja a ferramenta transferência fácil de arquivos e, depois, carregá-los nas pastas compartilhadas no servidor Windows Server Essentials em um único lote. Para obter mais informações, consulte o blog postar [compreensão remoto da Web acesso compartilhamento de arquivos](http://blogs.technet.com/b/sbs/archive/2012/04/19/understanding-remote-web-access-file-sharing.aspx).  
  
##  <a name="BKMK_7"></a>Exibir e procurar mídia digital compartilhada  
 Você pode exibir ou procurar recursos usando o painel, da barra inicial, o site de acesso via Web remoto ou o aplicativo meu servidor para Windows Phone.  
  
#### <a name="to-view-and-browse-shared-media-from-the-dashboard"></a>Para exibir e procurar mídia compartilhada do painel  
  
1.  Abra o painel do servidor.  
  
2.  Na barra de navegação principal, clique em **armazenamento**.  
  
3.  Clique no **pastas de servidor** guia. Aparece uma lista de pastas do servidor.  
  
4.  Clique duas vezes em uma pasta para exibir o conteúdo da pasta.  
  
#### <a name="to-view-and-browse-shared-media-using-the-launchpad-from-a-network-computer"></a>Para exibir e procurar mídia compartilhada usando a barra inicial de um computador de rede  
  
1.  Faça logon barra inicial.  
  
2.  Clique em **pastas compartilhadas**. Windows Explorer abre e exibe uma lista de pastas compartilhadas no servidor.  
  
3.  Clique duas vezes em uma pasta para exibir o conteúdo da pasta.  
  
#### <a name="to-view-and-browse-shared-media-using-remote-web-access"></a>Para exibir e procurar mídia compartilhada usando o acesso via Web remoto  
  
1.  Fazer logon em acesso via Web remoto.  
  
2.  Clique em **pastas compartilhadas**. O **pastas compartilhadas** seção da página da web exibe uma lista de pastas compartilhadas no servidor.  
  
3.  Clique duas vezes em uma pasta para exibir o conteúdo da pasta.
