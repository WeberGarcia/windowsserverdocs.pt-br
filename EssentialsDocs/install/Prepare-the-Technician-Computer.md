# <a name="prepare-the-technician-computer"></a>Preparar o computador do técnico

>Aplica-se a: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

O computador do técnico contém as ferramentas e documentação na avaliação do Windows e o Kit de implantação (ADK), e é onde você personalizar a imagem do sistema operacional que você pretende implantar. Conclua o seguinte para preparar o computador do técnico.  
  
-   Prepare um computador com o Windows 7, Windows 8 ou Windows 10.  
  
-   Instalar o Windows ADK no computador executando o Windows 7, Windows 8 ou Windows 10 do [site da Microsoft ](https://go.microsoft.com/fwlink/?LinkID=248647).
