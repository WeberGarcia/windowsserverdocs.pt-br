# <a name="remote-access-server-role-documentation"></a>Documentação de função servidor de acesso remoto

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016

Documentação de função de servidor acesso remota fornece instruções para implantar qualquer um dos serviços de função de três (VPN Always On, roteamento e serviço de acesso remoto, Proxy de aplicativo Web) individualmente ou no mesmo servidor. Por exemplo, esses documentos se aplicam às situações onde você implantou a qualquer combinação de serviços de função de três, como a implantação de RRAS e DirectAccess no mesmo servidor.
