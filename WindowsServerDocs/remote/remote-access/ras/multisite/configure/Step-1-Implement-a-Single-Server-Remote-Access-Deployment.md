# <a name="step-1-implement-a-single-server-remote-access-deployment"></a>Etapa 1 de implementar uma implantação de acesso remoto de servidor único

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016

A primeira etapa de configuração para implantar o acesso remoto em uma topologia de multissite é implementar uma implantação avançada de servidor único e, em seguida, planeje adicionar servidores para cada ponto de entrada multissite.  
  
## <a name="BKMK_1.1"></a>Implementar uma implantação de servidor único  
Antes de configurar uma implantação multissite, você deve configurar avançado único servidor de implantação do acesso remoto conforme descrito em [implantar um único servidor DirectAccess com configurações avançadas](https://technet.microsoft.com/windows-server-docs/networking/remote-access/directaccess/single-server-advanced/deploy-a-single-directaccess-server-with-advanced-settings).
