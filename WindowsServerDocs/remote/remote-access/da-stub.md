# <a name="directaccess"></a>DirectAccess

>Aplica-se a: Windows Server (Canal Semestral), Windows Server 2016

No Windows Server 2016, **DirectAccess e VPN** é um serviço da função de servidor **Acesso Remoto**.

O DirectAccess permite a conectividade para usuários remotos para recursos de rede da organização sem a necessidade de conexões de VPN (rede virtual privada) tradicionais. 

A documentação do DirectAccess agora está localizada na seção [Gerenciamento de acesso remoto e de servidor](../index.yml) do sumário do Windows Server 2016, em [Acesso remoto](./remote-access.md). Para obter mais informações, consulte [DirectAccess](directaccess/DirectAccess.md).
