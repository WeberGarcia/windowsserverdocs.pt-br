# <a name="step-8-snapshot-the-directaccess-cluster-nlb-configuration"></a>ETAPA 8 instantâneo da configuração de Cluster NLB DirectAccess

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016

Isso conclui o laboratório de teste do DirectAccess. Para salvar essa configuração para que você possa retornar rapidamente a um funcionamento do DirectAccess com a configuração de cluster NLB do qual você pode testar outros guias de laboratório de teste modular do DirectAccess, extensões de guia de laboratório de teste ou para seu próprio aprendizado e experimentação, faça o a seguir:  
  
1.  Em todos os computadores físicos ou máquinas virtuais do laboratório de teste, feche todas as janelas e realize um desligamento normal.  
  
2.  Se o laboratório for baseado em máquinas virtuais, salve um instantâneo de cada máquina virtual e nomeie o cluster do DirectAccess de instantâneos e o NLB. Se o seu laboratório usa computadores físicos, crie imagens de disco para salvar a configuração de laboratório de teste do DirectAccess.
