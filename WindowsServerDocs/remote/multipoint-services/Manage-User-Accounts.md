# <a name="manage-user-accounts"></a>Gerenciar contas de usuário
Esta seção descreve os diferentes tipos de contas de usuário, como criar contas de usuário e como gerenciar contas de usuário. Em um sistema de serviços do MultiPoint, há dois tipos de contas de usuário: *contas de usuário padrão* e *contas de usuário administrativo*, conforme descrito abaixo.

> [!TIP]
> O tópico [Considerações sobre conta de usuário](User-Account-Considerations.md) oferece diretrizes que você deve considerar ao criar e gerenciar contas de usuário.
