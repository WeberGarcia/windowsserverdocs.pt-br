# <a name="install-software-on-your-multipoint-services-system"></a>Instalar o software em seu sistema MultiPoint Services
Quando você estiver conectado como usuário administrativo, você pode instalar novos programas, no modo de console ou, de uma estação, no modo de estação. No entanto, é recomendável que você instale programas no modo de console.  
  
Você pode instalar novo software no computador executando o MultiPoint Server para que todos os usuários possam executar o software ou para que somente você pode usar o software, dependendo da instalação e as opções do software de licenciamento.  
   
1.  Faça logon no computador do MultiPoint Services como um administrador.  
  
2.  Abra o MultiPoint Manager.  
  
3.  Clique o **página inicial** guia e, em seguida, clique em **alterne para modo de console**.  
  
4.  Faça logon como administrador e instalar seus aplicativos.  
  
5.  Depois de concluir a instalação de aplicativos, retorne o computador para o modo de estação. Para fazer isso, no **página inicial** , clique em **alterne para modo de estação**.
