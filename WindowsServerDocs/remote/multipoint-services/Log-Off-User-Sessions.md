# <a name="log-off-user-sessions"></a>Fazer logoff das sessões do usuário
Os usuários padrão, os usuários do MultiPoint Dashboard e os usuários administrativos podem fazer logon e sair de suas sessões da área de trabalho como fariam com qualquer sessão do Windows. Além disso, os usuários administrativos e usuários do MultiPoint Dashboard podem encerrar as sessões de usuário em todas as sessões monitoradas no sistema MultiPoint Services.  
  
1.  No MultiPoint Dashboard, clique no **Home** guia.  
  
2.  Siga um destes procedimentos:  
  
    -   Para fazer logoff de uma sessão de usuário único ou sessões selecionadas, clique na imagem em miniatura da sessão que você deseja encerrar e, em seguida, clique no menu de lista suspensa do canto superior esquerdo. Clique em **Fazer logoff de usuários** e, em seguida, clique em **Fazer logoff de usuários selecionados**. Você também pode ver essa opção clicando com o botão direito do mouse nas miniaturas selecionadas.
  
    -   Para fazer logoff de todas as sessões de usuário, clique no menu de lista suspensa do canto superior esquerdo, clique em **fazer logoff de usuários**e, em seguida, clique em **fazer logoff de usuários todos os**.
