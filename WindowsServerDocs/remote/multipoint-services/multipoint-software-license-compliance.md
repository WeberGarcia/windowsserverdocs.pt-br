# <a name="important-information-about-software-license-compliance"></a>Informações importantes sobre a conformidade de licenças de software
Você deve estar em conformidade com os termos de licença para os programas que você usa no sistema MultiPoint Services. Você também deve ter o número adequado de licenças para o número de *estações* que você adicionar ao seu sistema MultiPoint Services, conforme descrito nas seções a seguir.  
  
## <a name="software-license-compliance"></a>Conformidade de licença de software  
Se você estiver usando outros programas da Microsoft ou todos os programas de terceiros com este software da Microsoft ou de serviço, os termos que vêm com programas se aplicam ao uso deles. Microsoft pode considerar qualquer falha em cumprir com esses termos também ser uma violação da licença da Microsoft.  
  
## <a name="client-access-license-cal-compliance"></a>Conformidade de licença (CAL) de acesso do cliente  
Se você quiser adicionar uma estação para o seu sistema MultiPoint Services, você deve proteger o número apropriado de *licenças de acesso para cliente (CALs)* para todas as estações conectadas ao MultiPoint Services.
