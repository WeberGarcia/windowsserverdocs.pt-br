# <a name="enable-file-sharing-in-multipoint-services"></a>Habilitar compartilhamento de arquivos no MultiPoint Services
Você pode permitir que os usuários em suas estações MultiPoint de compartilhamento de arquivos de duas maneiras:  
  
-   **Se você tiver um servidor de arquivos na rede,** é recomendável que você crie uma pasta compartilhada no servidor de arquivos.  
  
-   **Se você tiver uma rede pequena de servidores de MultiPoint de 2 a 3, com nenhum servidor de arquivos dedicado,** um dos servidores do MultiPoint pode agir como o servidor de arquivos para todos os computadores restantes executando o MultiPoint services. Criar uma pasta compartilhada no servidor e, em seguida, criar contas de usuário local para todos os usuários no servidor. A pasta compartilhada pode estar na unidade interna do original, ou você pode anexar outras unidades internas ou externas ao computador.
