# <a name="remap-all-stations"></a>Remapear Todas as Estações
O remapeamento de estações permite que você associe teclados e mouses a monitores. Quando você remapear todas as estações, as configurações originais, como informações de nome e logon automático, são apagadas. Todas as estações de usuário local são suspensas enquanto o remapeamento ocorre.  
  
1.  Abra o MultiPoint Manager no modo de estação e, em seguida, clique no **Home** guia.  
2.  Em **Tarefas**, clique **Remapear todas as estações**.  
3. Siga as instruções nas telas de estação para associar os teclados às estações em seu sistema.
