# <a name="install-updates"></a>Instalar atualizações
É recomendável que você instale atualizações, se disponível. Instalando atualizações requer uma conexão de Internet.  

1.  Na tela **Inicial** , abra o **Painel de Controle**.  
  
2.  No painel de controle, digite **atualizações**e, em seguida, clique em **verificar atualizações**.  
  
3.  Se o site do Windows Update listar todas as atualizações que são necessários em seu computador, instale as atualizações.
