# <a name="prepare-your-multipoint-services-system-for-users"></a>Preparar seu sistema MultiPoint Services para usuários
Depois de instalar e configurar o MultiPoint Services e executar qualquer configuração adicional e configurações de hardware, você está pronto para dar aos usuários acesso ao sistema. Você precisará planejar e criar contas de usuário. Em alguns ambientes, você também precisa configurar estações para logon automático e permite várias sessões para suas contas de usuário compartilhadas. E você precisa decidir como configurar o compartilhamento de arquivos para seus usuários. Todos esses tópicos são abordados nesta seção.  
  
> [!NOTE]  
> Depois de criar suas contas de usuário e fazer outras atualizações de configuração para se preparar para os usuários, é recomendável que você ative a proteção de disco para que nenhum usuário inadvertidamente pode fazer alterações às configurações e arquivos do sistema. Para obter mais informações, consulte [configurar a proteção de disco](Configure-Disk-Protection-in-MultiPoint-services.md).
