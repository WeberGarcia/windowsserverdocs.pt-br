# <a name="remote-desktop-services---connect-from-any-device"></a>Serviços de área de trabalho remota - conectar de qualquer dispositivo

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016

Acesse recursos corporativos de qualquer Windows, Apple ou Android computador, tablet ou telefone. Habilite usuários a ver com facilidade suas áreas de trabalho disponíveis e aplicativos de qualquer dispositivo por meio do Feed de Web da área de trabalho remota.

Saiba mais sobre [clientes de área de trabalho remota Microsoft](clients/remote-desktop-clients.md).
