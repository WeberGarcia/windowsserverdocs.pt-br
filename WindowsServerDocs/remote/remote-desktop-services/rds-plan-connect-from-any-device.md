# <a name="remote-desktop-services---connect-from-any-device"></a>Serviços de Área de Trabalho Remota – Conectar de qualquer dispositivo

>Aplica-se a: Windows Server (Canal Semestral), Windows Server 2019, Windows Server 2016

Acesse recursos corporativos de qualquer telefone, tablet ou computador Windows, Apple ou Android. Habilite usuários a ver com facilidade suas áreas de trabalho e aplicativos disponíveis de qualquer dispositivo por meio do Feed de Web da Área de Trabalho Remota.

Saiba mais sobre [Clientes de Área de Trabalho Remota da Microsoft](clients/remote-desktop-clients.md).
