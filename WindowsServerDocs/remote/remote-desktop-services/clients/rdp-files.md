# <a name="supported-remote-desktop-rdp-file-settings"></a>Suporte para configurações de arquivo RDP da área de trabalho remota

A tabela a seguir inclui a lista de configurações de arquivo RDP com suporte que você pode usar com os clientes do Windows e HTML. Um "x" na coluna de plataforma indica que a configuração tem suporte. No entanto, essa lista não é uma lista completa de configurações com suporte para os clientes Windows e o HTML5. Continuaremos a atualizar esta tabela para incluir configurações de RDP mais com suporte para os clientes do Windows e o HTML5, bem como o macOS, iOS e Android clientes.

Essa configuração corresponde ao caminho e arquivo de caixa de nome programa na guia Programas do cliente da Conexão de área de trabalho remota. Para especificar um shell alternativo, insira um caminho válido para um arquivo executável para o valor, como "C:\ProgramFiles\Office\word.exe". Essa configuração também determina qual caminho ou o alias do aplicativo remoto a ser iniciado em tempo de conexão se Remoteapplicationmode=i:&lt;0 está habilitado. | "C:\ProgramFiles\Office\word.exe" || x | x | x |
