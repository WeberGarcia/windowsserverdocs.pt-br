# Configurações de privacidade de aplicativos gerenciados e áreas de trabalho

>Aplica-se a: Windows 10, Windows 8.1, Windows Server 2012 R2, Windows Server 2016

Ao acessar recursos gerenciados (aplicativos ou desktops) fornecidos pelo administrador de TI, as configurações de privacidade para o sistema remoto tiverem sido pré-configurados pelo administrador de TI. Essas configurações podem ser diferentes das configurações de privacidade em seu sistema local. 

>[!NOTE]
>Usar recursos gerenciados em regiões diferentes Estados Unidos pode resultar em transferência de dados para os Estados Unidos.

Estas são algumas das configurações de privacidade do Windows 10 que você pode configurar em sua área de trabalho gerenciada:
- [Reconhecimento de fala](https://go.microsoft.com/fwlink/?linkid=874646)
- [Localizar meu dispositivo](https://go.microsoft.com/fwlink/?linkid=533063)
- [Escrita à tinta & digitação](https://go.microsoft.com/fwlink/?linkid=874646)
- [ID de Anúncio](https://go.microsoft.com/fwlink/?linkid=838419)
- [Localização](https://go.microsoft.com/fwlink/?linkid=529987)
- [Dados de diagnóstico](https://go.microsoft.com/fwlink/?linkid=614828)
- [Experiências personalizadas](https://go.microsoft.com/fwlink/?linkid=614828)

Você sempre pode revisar as informações coletadas e enviados à Microsoft acessando seu [Painel de privacidade](https://go.microsoft.com/fwlink/?linkid=864206).

## Saiba mais sobre as configurações de privacidade

Se o administrador de TI tiver fornecido uma área de trabalho gerenciada, você pode seguir as instruções na próxima seção para saber mais sobre essas configurações e alterar as configurações que não são bloqueadas pelo administrador de TI.

### Como alterar as configurações de privacidade em áreas de trabalho remotas do Windows 10

Para alterar as configurações de privacidade em uma área de trabalho remota do Windows 10:

1. Na área de trabalho remota, selecione o botão Windows na barra de tarefas ou pressione a tecla Windows no teclado para abrir o menu Iniciar.
2. Selecione o ícone de engrenagem para abrir as configurações.
3. Procure os nomes das configurações de privacidade configurável listados anteriormente neste tópico para saber mais sobre ele.

>[!NOTE]
> Se o administrador de TI tiver configurado a área de trabalho gerenciada para reter não definições de configuração do usuário entre as conexões, as alterações feitas para essas configurações não serão salvas.
