# <a name="additional-remote-desktop-resources"></a>Recursos adicionais da área de trabalho remota

Além das informações aqui na biblioteca do Windows Server 2016, você pode usar os recursos a seguir para saber mais sobre e obter ajuda com os serviços de área de trabalho remota:

- No Twitter? Portanto, estamos: [@RDS4U](https://twitter.com/RDS4U)
- Participe de discussões sobre os serviços de área de trabalho remota no gerais a [Fórum do TechNet RDS](https://aka.ms/technetforum-rds).
- Para discussão sobre aplicativos/clientes de área de trabalho remota para Windows, Android, iOS e Mac, visite o [Fórum do TechNet de clientes de área de trabalho remota](https://aka.ms/technetforum-rdc).
- Para vários pontos, confira a [Fórum do TechNet do MultiPoint](https://aka.ms/multipoint-forum).

Se você tiver ideias sobre os serviços de área de trabalho remota que você deseja compartilhar conosco, postar um tópico na [nosso fórum UserVoice](https://aka.ms/uservoice-rds).
