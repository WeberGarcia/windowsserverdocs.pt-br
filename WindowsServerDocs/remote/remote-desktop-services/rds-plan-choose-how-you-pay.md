# <a name="remote-desktop-services---choose-how-you-pay"></a>Serviços de Área de Trabalho Remota – Escolha como pagar

>Aplica-se a: Windows Server (Canal Semestral), Windows Server 2019, Windows Server 2016

Escolha seu licenciamento com base no que faz sentido para sua empresa. Licença por usuário para permitir usuários remotos em qualquer um dos dispositivos em um cenário BYOD. Licença por dispositivo se os usuários compartilharem os mesmos dispositivos. Se você for um provedor de serviços (HSP ou MSP) ou ISV, escolha a licença SALs por usuário para um modelo pago conforme o uso e flexível.

Para obter mais informações, confira [Licença de sua implantação do RDS com licenças de acesso para cliente (CALs)](rds-client-access-license.md).
