# <a name="remote-desktop-services---choose-how-you-pay"></a>Serviços de área de trabalho remota - escolha como você paga

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016

Escolha seu licenciamento com base no que faz sentido para sua empresa. Licença por usuário para permitir que usuários remotos em qualquer um dos seus dispositivos em um cenário BYOD. Licença por dispositivo, se os usuários compartilharem os mesmos dispositivos. Se você for um provedor de serviço (HSP ou MSP) ou ISV, escolha a licença por usuário SALs para um modelo pago conforme o uso e flexível.

Para obter mais informações, confira [licença de sua implantação do RDS com licenças de acesso para cliente (CALs)](rds-client-access-license.md).
