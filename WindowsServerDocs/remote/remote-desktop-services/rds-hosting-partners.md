# <a name="remote-desktop-services-hosting-partners-and-assessment"></a>Avaliação e parceiros de hospedagem de serviços de área de trabalho remota

Recentemente, a Microsoft lançou um novo caminho de aprendizado na rede de parceiros da Microsoft: "Hospedagem de área de trabalho do Windows e aplicativos usando os serviços de área de trabalho remota no Azure".

Se você for um parceiro da Microsoft e deseja incluir na lista de parceiros que passaram a avaliação, aqui estão as etapas que você pode tomar para concluir o roteiro de aprendizagem:

1. Se tornar um [Microsoft Partner](https://partner.microsoft.com/), se você não ainda estiver.
2. Assista a [hospedando o Windows e aplicativos usando os serviços de área de trabalho remota na sessão de treinamento do Azure](https://mspartnerlp.partner.microsoft.com/LearningPath/LearningPath/DLPaths?trackId=2915&rowId=3603).
3. Executar o [avaliação técnica](https://mspartnerlp.partner.microsoft.com/LearningPath/LearningPath/DLPaths?trackId=1660&rowId=2220&trackPathId=9871).
4. Verifique se você atende a [requisitos para a competência de plataforma de nuvem](https://partner.microsoft.com/en-us/membership/cloud-platform-competency).

Já um Microsoft Partner e dúvidas? Contate a equipe de área de trabalho remota em <rdhostingpartners@microsoft.com>.  


## <a name="partners-who-have-passed-the-learning-path-assessment"></a>Parceiros que passaram a aprendizagem avaliação de caminho 

Se você for um cliente procurando por um parceiro para ajudá-lo a áreas de trabalho do host Windows e aplicativos no Azure para seus usuários, podemos compilou uma lista de parceiros que passaram a avaliação. Aqui está uma [lista de parceiros](https://github.com/MicrosoftDocs/windowsserverdocs/blob/master/WindowsServerDocs/remote/remote-desktop-services/RDS-Hosting-Partners.pdf), de 28/03/2017, que você pode baixar.

Você pode encontrar mais informações sobre cada parceiro usando as seguintes etapas:

1. Abra [encontre um parceiro](https://partnercenter.microsoft.com/pcv/search).
2. Desmarque a **local** campo.
3. Insira o nome do parceiro na **que estou procurando ajuda com** campo.
