# <a name="features-removed-or-planned-for-replacement-in-windows-server"></a>Recursos removidos ou com substituição planejada no Windows Server

> Aplica-se a: Windows Server 2019, Windows Server 2016, Windows Server (canal semestral)

Você pode encontrar os recursos removidos e preteridos para cada uma dessas versões do Windows Server.  

Versões do LTSC (Canal de Manutenção em Longo Prazo):

- [Recursos removidos ou com remoção planejada no Windows Server 2019](removed-features-19.md)
- [Recursos removidos ou preteridos no Windows Server 2016](../get-started/Deprecated-Features.md)

Versões do Canal Semestral:

- [Recursos removidos ou com substituição planejada do Windows Server versão 1903 em diante](removed-features-1903.md)
- [Recursos removidos ou com substituição planejada a partir do Windows Server, versão 1803](../get-started/windows-server-1803-removed-features.md)
- [Recursos removidos ou com substituição planejada do Windows Server versão 1709 em diante](../get-started/Removed-Features-1709.md) (sem suporte desde 9 de abril de 2019)

As versões do Canal semestral do Windows Server fornecem novos recursos de sistema operacional em um ritmo mais rápido. Uma nova versão é lançada a cada seis meses e tem suporte por 18 meses. Verifique as [Informações sobre versão do Windows Server](../get-started/windows-server-release-info.md) e a [Página de suporte do ciclo de vida](https://support.microsoft.com/lifecycle) para datas de suporte e use sempre a versão mais recente, se possível.
