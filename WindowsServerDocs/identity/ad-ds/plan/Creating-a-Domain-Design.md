# <a name="creating-a-domain-design"></a>Criar um design de domínio

>Aplica-se a: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

O proprietário da floresta é responsável por criar um design de domínio da floresta. Criar um design de domínio envolve examinando os requisitos de replicação e a capacidade existente de sua infraestrutura de rede e, em seguida, criar uma estrutura de domínio que permite que os serviços de domínio do Active Directory (AD DS) para funcionar da maneira mais eficiente. Domínios são usados para a partição do diretório para que as informações no diretório podem ser distribuídas e gerenciadas com eficiência em toda a empresa. A meta para o design de domínio é maximizar a eficiência do Active Directory, topologia de replicação, garantindo que a replicação não usa muita largura de banda de rede disponível e não interfere na operação diária de sua rede.
