# <a name="deploying-ad-ds-in-a-new-organization"></a>Implantando o AD DS em uma nova organização

>Aplica-se a: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Design completamente Preparando seus serviços de domínio Active do Directory (AD DS) é essencial para uma implantação econômica. Se o seu ambiente de rede está funcionando no momento sem um serviço de diretório, realizar um design abrangente da estrutura lógica do AD DS antes de implantar o AD DS. Em seguida, você pode implantar um novo domínio raiz e implante o resto da estrutura do domínio acordo com seu design.  
  
A ilustração a seguir mostra as etapas para implantar o Windows Server 2008 AD DS em um ambiente de rede que atualmente está operando sem um serviço de diretório.  
  
![Implantar em uma novo organograma](media/Deploying-AD-DS-in-a-New-Organization/daa38971-86f2-4033-9442-0cdff9ecc48f.gif)  
  
Para obter uma lista de tarefas detalhadas que você pode usar para planejar e implantar o AD DS em uma nova organização, consulte [lista de verificação: Implantando o AD DS em uma nova organização](https://technet.microsoft.com/library/cc725897.aspx).
