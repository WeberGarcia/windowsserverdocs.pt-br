# <a name="ad-ds-troubleshooting"></a>Solução de problemas do AD DS

>Aplica-se a: Windows Server 2016, Windows Server 2012 R2

Esta seção inclui procedimentos para diagnosticar e recomendações de solução de problemas e corrigir problemas que pode ocorrer com a replicação do Active Directory.

Este conteúdo se concentra principalmente nas respostas a mensagens de log de eventos do serviço de diretório e mensagens de erro com base em ferramenta que podem ser relatadas pelas ferramentas Repadmin.exe e Dcdiag.exe. Essas ferramentas estão disponíveis em todos os controladores de domínio que estão executando o Windows Server 2016 ou 2012 R2. Você também pode instalar ferramentas de administração de servidor remoto (RSAT) em um servidor membro que está executando o Windows 10.

Para obter informações sobre como instalar o RSAT, consulte o artigo [ferramentas de administração de servidor remoto](https://docs.microsoft.com/windows-server/remote/remote-server-administration-tools).
