# <a name="azure-active-directory-connect"></a>Azure Active Directory Connect


Azure AD Connect integrará seus diretórios locais com Azure Active Directory. Isso permite que você forneça uma identidade comum para seus usuários para aplicativos do Office 365, do Azure e SaaS integrados ao Azure AD. .  
  
Para obter mais informações, consulte [integrando suas identidades locais com o Azure Active Directory](https://azure.microsoft.com/documentation/articles/active-directory-aadconnect/).
