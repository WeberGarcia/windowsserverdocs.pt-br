# <a name="checklist-configuring-the-resource-partner-organization"></a>Lista de verificação: Configurando a organização do parceiro de recurso

>Aplica-se a: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

A organização de parceiro de recurso contém os servidores Web hospeda os aplicativos baseados em Web\ que serão acessados por usuários no parceiro de conta. Os administradores na organização devem usar o AD FS snap\-in Gerenciamento para criar relações de confiança de provedor de declarações para representar suas relações de confiança com organizações parceiras de conta. Por sua vez, o administrador de parceiro de conta deve criar terceira relações de confiança de terceiros para cada organização do parceiro de conta que desejam confiar.  
  
Esta lista de verificação inclui as tarefas que são necessárias para implantar os serviços de Federação do Active Directory \(AD FS\) na organização do parceiro de recurso. Ele também inclui tarefas para configurar os componentes que são necessárias para estabelecer one\-metade de uma parceria de Federação.  
  
Se você estiver implantando um [Web SSO Design](https://technet.microsoft.com/library/dd807033.aspx), você não precisa seguir essa lista de verificação. No entanto, você precisa concluir as tarefas nesta lista de verificação para implantar com êxito um [Design de SSO da Web federado](https://technet.microsoft.com/library/dd807050.aspx).  
  
> [!IMPORTANT]  
> Certifique-se de que o administrador da organização do parceiro de conta segue as diretrizes em [lista de verificação: Configurando a organização do parceiro de conta](Checklist--Configuring-the-Account-Partner-Organization.md) para garantir que todas as tarefas de implantação necessárias serão efetuadas para criar com êxito a segunda metade da parceria federação  
  
> [!NOTE]  
> Conclua as tarefas nesta lista de verificação na ordem. Quando um link de referência leva você para um procedimento, retorne para este tópico depois de concluir as etapas no procedimento para que você pode continuar com as tarefas restantes desta lista de verificação.
