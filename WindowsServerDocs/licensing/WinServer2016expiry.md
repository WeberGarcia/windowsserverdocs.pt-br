# <a name="windows-server-technical-preview-expiration"></a>Expiração do Windows Server Technical Preview
Cada uma dessas versões do Windows Server 2016 Technical Preview contém hora\-software confidencial. Ao atingir a data de expiração, você deixa de ter autorização para usar a versão.  

## <a name="license-expiration-dates-for-technical-previews"></a>Datas de expiração de licença para Technical Previews  
Esta tabela detalha a data de expiração de licença para cada uma das versões do Windows Server 2016 Technical Preview.  

|Versão|Data de liberação|Data de validade da licença|  
|-----------|----------------|---------------------------|  
|Windows Server " 2016 Technical Preview 3|Agosto de 2015|1º de agosto de 2016|  
|Windows Server " 2016 Technical Preview 4|Novembro de 2015|15 de outubro de 2016|  
|Windows Server " 2016 Technical Preview 5|Abril de 2016|28 de fevereiro de 2017|
