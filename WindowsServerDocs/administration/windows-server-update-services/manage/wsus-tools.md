 ## <a name="wsus-reporting-rollup-sample-toolhttpsdownloadmicrosoftcomdownload339339ac5ee-ae9a-44a4-b09c-483736294433wsusrollupsampleexe"></a>[Ferramenta de exemplo do pacote cumulativo de atualizações do WSUS Reporting](https://download.microsoft.com/download/3/3/9/339ac5ee-ae9a-44a4-b09c-483736294433/WSUSRollupSample.EXE)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    
 > Essa ferramenta usa a interface de programação de aplicativo (API) do WSUS para demonstrar centralizado, monitoramento e relatórios do WSUS. Ele cria um único relatório de status de atualização e de computador dos servidores WSUS em seu ambiente WSUS. O pacote de exemplo também contém os arquivos de origem de exemplo para personalizar ou estender a funcionalidade da ferramenta da ferramenta para atender às necessidades específicas. Os arquivos e relatar a ferramenta de exemplo de pacote cumulativo de atualizações do WSUS são fornecidos como estão. Não há suporte do produto está disponível para arquivos de exemplo ou da ferramenta. Para obter mais informações, consulte o [arquivo Leiame](https://download.microsoft.com/download/8/1/a/81a41962-cff5-4396-a567-0d2f87d8f67a/Readme.htm).
