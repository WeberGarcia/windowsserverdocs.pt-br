# <a name="create-partition-primary"></a>create partition primary

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Cria uma partição primária no disco básico com foco.  
  
> [!CAUTION]  
  
  
  
## <a name="syntax"></a>Sintaxe  
  
```  
create partition primary [size=<n>] [offset=<n>] [id={ <byte> | <guid> }] [align=<n>] [noerr]  
```

## <a name="remarks"></a>Comentários  
  
-   Depois de criar a partição, o foco mudará automaticamente para a nova partição.  
  
-   A partição não recebe uma letra da unidade. Você deve usar o comando **assign** no DiskPart para atribuir uma letra de unidade à partição.  
  
-   Um disco básico deve ser selecionado para que essa operação tenha sucesso. Use o comando **selecionar disco** para selecionar um disco básico e deslocar o foco para ele.  
  
## <a name="BKMK_examples"></a>Disso  
Para criar uma partição primária de 1000 megabytes de tamanho, digite:  
  
```  
create partition primary size=1000  
```
