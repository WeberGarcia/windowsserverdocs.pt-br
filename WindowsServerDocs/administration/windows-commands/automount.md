# <a name="automount"></a>montagem automática



Habilita ou desabilita o recurso de montagem automática. Quando habilitado (o padrão), o Windows monta automaticamente o sistema de arquivos para um novo volume básico ou dinâmico quando ele é adicionado ao sistema e atribui uma letra de unidade ao volume.
