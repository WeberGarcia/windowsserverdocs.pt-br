# <a name="automount"></a>Montagem automática



Habilita ou desabilita o recurso de montagem automática. Quando habilitado (padrão), o Windows monta automaticamente o sistema de arquivos para um novo volume básico ou dinâmico quando ele é adicionado ao sistema e, em seguida, atribui uma letra de unidade ao volume.
