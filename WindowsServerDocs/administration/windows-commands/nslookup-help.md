# <a name="nslookup-help"></a>nslookup help

Exibe o texto de ajuda de subcomando.

## <a name="syntax"></a>Sintaxe

```
help
```

```
?
```

### <a name="parameters"></a>Parâmetros

| Parâmetro | Descrição |
| --------- | ----------- |
| /? | Exibe a ajuda no prompt de comando. |
| /help | Exibe a ajuda no prompt de comando. |
