# <a name="help"></a>ajuda



Fornece informações online sobre comandos do sistema (ou seja, comandos que não são de rede). Se usado sem parâmetros, lista de **ajuda** e descreve brevemente todos os comandos do sistema.

Para obter exemplos de como usar esse comando, consulte [Exemplos](#BKMK_examples).

## <a name="syntax"></a>Sintaxe

```
help [<Command>] 
[<Command>] /?
```

### <a name="parameters"></a>Parâmetros

|Parâmetro|Descrição|
|---------|-----------|
|> de comando \<|Especifica o nome do comando sobre o qual você deseja obter informações.|

## <a name="examples"></a><a name=BKMK_examples></a>Disso

Para exibir informações sobre o comando **Robocopy** , digite um dos seguintes:
```
help robocopy
robocopy /? 
```
