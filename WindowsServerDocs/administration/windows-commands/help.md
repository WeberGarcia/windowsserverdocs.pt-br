# <a name="help"></a>ajuda



Fornece informações online sobre os comandos do sistema (ou seja, comandos de fora da rede). Se usado sem parâmetros, **ajudar** lista e descreve resumidamente cada comando do sistema.

Para obter exemplos de como usar esse comando, consulte [Exemplos](#BKMK_examples).

## <a name="syntax"></a>Sintaxe

```
help [<Command>] 
[<Command>] /?
```

## <a name="parameters"></a>Parâmetros

|Parâmetro|Descrição|
|---------|-----------|
|\<Comando >|Especifica o nome do comando que você deseja obter informações sobre.|

## <a name="BKMK_examples"></a>Exemplos

Para exibir informações sobre o **robocopy** comando, digite o seguinte:
```
help robocopy
robocopy /? 
```
