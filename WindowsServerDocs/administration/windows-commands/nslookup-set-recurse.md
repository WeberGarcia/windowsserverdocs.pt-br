# <a name="nslookup-set-recurse"></a>nslookup set recurse



Informa ao servidor de nome de sistema de nome de domínio (DNS) para consultar outros servidores se não tiver as informações.

## <a name="syntax"></a>Sintaxe

```
set [no]recurse
```

## <a name="parameters"></a>Parâmetros

|   Parâmetro   |                                                                  Descrição                                                                  |
|---------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| **norecurse** |                Interrompe o servidor de nomes de consultar outros servidores se não tiver as informações do sistema de nome de domínio (DNS).                |
|  **recurse**  | Informa ao servidor de nome de sistema de nome de domínio (DNS) para consultar outros servidores se não tiver as informações. A sintaxe padrão é **recurse**. |
|     {Ajuda     |                                                                      ?}                                                                       |
