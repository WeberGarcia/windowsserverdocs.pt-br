# <a name="nslookup-set-recurse"></a>nslookup set recurse



Informa ao servidor de nomes DNS (sistema de nomes de domínio) para consultar outros servidores se ele não tiver as informações.

## <a name="syntax"></a>Sintaxe

```
set [no]recurse
```

## <a name="parameters"></a>Parâmetros

|   Parâmetro   |                                                                  Descrição                                                                  |
|---------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| **norepetir** |                Interrompe o servidor de nomes DNS (sistema de nomes de domínio) de consultar outros servidores se ele não tiver as informações.                |
|  **recurse**  | Informa ao servidor de nomes DNS (sistema de nomes de domínio) para consultar outros servidores se ele não tiver as informações. A sintaxe padrão é **recurse**. |
