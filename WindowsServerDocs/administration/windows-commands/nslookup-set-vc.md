# <a name="nslookup-set-vc"></a>nslookup set vc



Especifica se deve ou não usar um circuito virtual ao enviar solicitações ao servidor.

## <a name="syntax"></a>Sintaxe

```
set [no]vc
```

## <a name="parameters"></a>Parâmetros

|Parâmetro|Descrição|
|---------|-----------|
|**novc**|Especifica a nunca use um circuito virtual ao enviar solicitações ao servidor. O padrão é **novc**.|
|**vc**|Especifica para usar sempre um circuito virtual ao enviar solicitações ao servidor.|
|{Ajuda | ?}|Exibe um resumo breve dos **nslookup** subcomandos.|
