# <a name="bitsadmin-peers-and-list"></a>bitsadmin peers e list

Lista todos os pares.

## <a name="syntax"></a>Sintaxe

```
bitsadmin /peers /list
```

## <a name="examples"></a>Exemplos

Para listar todos os pares:

```
bitsadmin /peers /list
```
