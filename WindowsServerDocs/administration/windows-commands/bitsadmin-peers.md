# <a name="bitsadmin-peers"></a>Bitsadmin pares

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

```
bitsadmin /Peers /help
bitsadmin /Peers /Discover
bitsadmin /Peers /Clear
bitsadmin /Peers /list
```
## <a name="parameters"></a>Parâmetros
|Parâmetro|Descrição|
|-------|--------|
|[Ajuda e colegas Bitsadmin](bitsadmin-peers-and-help.md)|Exibe o uso de linha de comando para as opções de /Peers.|
|[Bitsadmin pares e descobrir](bitsadmin-peers-and-discover.md)|Descobre pares novamente.|
|[pares de Bitsadmin e clear](bitsadmin-peers-and-clear.md)|Limpa a lista de colegas.|
|[pares de Bitsadmin e lista](bitsadmin-peers-and-list.md)|Liste todos os pares.|
