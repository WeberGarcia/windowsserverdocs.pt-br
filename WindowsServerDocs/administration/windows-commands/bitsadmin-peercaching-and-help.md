# <a name="bitsadmin-peercaching-and-help"></a>Bitsadmin e ajuda de cache



Exibe o uso de linha de comando para os comutadores de **cache** .

## <a name="syntax"></a>Sintaxe

```
Bitsadmin /PeerCaching /Help 
```

## <a name="BKMK_examples"></a>Disso

O comando a seguir exibe a ajuda de linha de comando para os comutadores de **serviço de cache** .
```
Bitsadmin /PeerCaching /Help
```
