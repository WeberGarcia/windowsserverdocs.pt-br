# <a name="bitsadmin-peercaching-and-help"></a>Ajuda e o cache par do bitsadmin



Exibe o uso de linha de comando para o **cache par** comutadores.

## <a name="syntax"></a>Sintaxe

```
Bitsadmin /PeerCaching /Help 
```

## <a name="BKMK_examples"></a>Exemplos

O comando a seguir exibe a Ajuda de linha de comando para o **cache par** comutadores.
```
Bitsadmin /PeerCaching /Help
```
