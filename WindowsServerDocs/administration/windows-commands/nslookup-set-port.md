# <a name="nslookup-set-port"></a>nslookup set port



Altera a porta do servidor de nome DNS (sistema de nomes de domínio) TCP/UDP padrão para o valor especificado.

## <a name="syntax"></a>Sintaxe

```
set port=<Port>
```

### <a name="parameters"></a>Parâmetros

| Parâmetro |                                          Descrição                                          |
|-----------|-----------------------------------------------------------------------------------------------|
|  \<> de porta  | Especifica o novo valor para a porta do servidor de nome DNS TCP/UDP padrão. A porta padrão é 53. |
|   {ajuda   |                                              ?}                                               |
