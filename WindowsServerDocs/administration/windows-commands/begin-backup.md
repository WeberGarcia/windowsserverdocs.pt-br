# <a name="begin-backup"></a>Iniciar backup

inicia uma sessão de backup completo.

## <a name="syntax"></a>Sintaxe

```
begin backup
```

## <a name="remarks"></a>Comentários

Esse comando substitui a configuração de backup de cópia padrão.
