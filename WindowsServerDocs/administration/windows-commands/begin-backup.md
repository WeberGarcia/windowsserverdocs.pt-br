# <a name="begin-backup"></a>Início do backup



Inicia uma sessão de backup completo.

## <a name="syntax"></a>Sintaxe

```
begin backup
```

## <a name="remarks"></a>Comentários

-   Esse comando substitui a configuração de backup de cópia do padrão.
