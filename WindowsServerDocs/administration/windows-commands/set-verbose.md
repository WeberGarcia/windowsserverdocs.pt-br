# <a name="set-verbose"></a>Conjunto detalhado



Especifica se a saída detalhada é fornecida durante a criação de cópias de sombra. Se usado sem parâmetros, **definido detalhado** exibe a Ajuda no prompt de comando.

## <a name="syntax"></a>Sintaxe

```
set verbose {on | off}
```

## <a name="parameters"></a>Parâmetros

|Parâmetro|Descrição|
|---------|-----------|
|{on | off}|Ativa ou desativa o modo detalhado.|

## <a name="remarks"></a>Comentários

-   Se o modo detalhado estiver ativado, **definir** fornece detalhes do gravador de inclusão ou exclusão e detalhes de compactação de metadados e a extração.
