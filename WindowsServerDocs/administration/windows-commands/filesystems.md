# <a name="filesystems"></a>sistemas de arquivos



Exibe informações sobre o sistema de arquivos atual do volume com o foco e lista os sistemas de arquivos que têm suporte para formatar o volume.

## <a name="syntax"></a>Sintaxe

```
filesystems
```

## <a name="remarks"></a>Comentários

-   Um volume deve ser selecionado para essa operação seja bem-sucedida. Use o **selecionar volume** comando para selecionar um volume e mudar o foco a ele.
