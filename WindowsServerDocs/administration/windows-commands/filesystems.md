# <a name="filesystems"></a>sistemas



Exibe informações sobre o sistema de arquivos atual do volume com foco e lista os sistemas de arquivos com suporte para formatar o volume.

## <a name="syntax"></a>Sintaxe

```
filesystems
```

## <a name="remarks"></a>Comentários

-   Um volume deve ser selecionado para que essa operação seja realizada com sucesso. Use o comando **selecionar volume** para selecionar um volume e deslocar o foco para ele.
