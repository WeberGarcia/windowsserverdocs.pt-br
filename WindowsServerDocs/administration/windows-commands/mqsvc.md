# <a name="mqsvc"></a>mqsvc



A tecnologia de enfileiramento de mensagens permite que os aplicativos executados em momentos diferentes se comuniquem em redes e sistemas heterogêneos que podem estar temporariamente offline. O enfileiramento de mensagens fornece entrega de mensagens garantida, roteamento eficiente, segurança e mensagens baseadas em prioridade. Ele pode ser usado para implementar soluções para cenários de mensagens assíncronas e síncronas. Para obter mais informações sobre esse comando, consulte [MSMQ (enfileiramento de mensagens)](https://go.microsoft.com/fwlink/?LinkId=248723) no msdn.

## <a name="syntax"></a>Sintaxe

```
Mqsvc.exe
```

### <a name="parameters"></a>Parâmetros

Nenhuma
