# <a name="mqsvc"></a>mqsvc

A tecnologia de enfileiramento de mensagens permite que os aplicativos em execução em momentos diferentes se comuniquem em redes e sistemas heterogêneos que podem estar temporariamente offline. O enfileiramento de mensagens fornece entrega de mensagens garantida, roteamento eficiente, segurança e mensagens baseadas em prioridade. Ele pode ser usado para implementar soluções para cenários de mensagens assíncronas e síncronas. Para obter mais informações sobre o enfileiramento de mensagens, consulte [MSMQ (enfileiramento de mensagens)](https://docs.microsoft.com/previous-versions/windows/desktop/legacy/ms711472(v=vs.85)).

## <a name="syntax"></a>Syntax

```
mqsvc.exe
```
