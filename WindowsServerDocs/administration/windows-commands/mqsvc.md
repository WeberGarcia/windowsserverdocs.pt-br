# <a name="mqsvc"></a>mqsvc



Tecnologia de enfileiramento de mensagens permite que os aplicativos executados em momentos diferentes para se comunicar por redes heterogêneas e sistemas que podem estar temporariamente offline. O Message Queuing fornece entrega de mensagens garantida, roteamento eficiente, segurança e sistema de mensagens baseado em prioridade. Ele pode ser usado para implementar soluções para cenários de mensagens síncronas e assíncronas. Para obter mais informações sobre esse comando, consulte [Message Queuing (MSMQ)](https://go.microsoft.com/fwlink/?LinkId=248723) no MSDN.

## <a name="syntax"></a>Sintaxe

```
Mqsvc.exe
```

### <a name="parameters"></a>Parâmetros

Nenhuma
