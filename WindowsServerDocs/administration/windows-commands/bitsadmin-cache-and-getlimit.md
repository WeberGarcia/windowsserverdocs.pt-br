# <a name="bitsadmin-cache-and-getlimit"></a>bitsadmin cache and getlimit

Recupera o limite de cache.

## <a name="syntax"></a>Sintaxe

```
bitsadmin /cache /getlimit
```

## <a name="examples"></a>Exemplos

Para recuperar o limite de cache:

```
bitsadmin /cache /getlimit
```
