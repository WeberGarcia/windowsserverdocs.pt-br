# <a name="bitsadmin-cache-and-clear"></a>bitsadmin cache and clear

Limpa o cache local.

## <a name="syntax"></a>Syntax

```
bitsadmin /cache /clear
```

## <a name="examples"></a>Exemplos

Para limpar o cache local:

```
bitsadmin /cache /clear
```
