# <a name="bitsadmin-cache-and-clear"></a>cache bitsadmin e limpe



Limpa o cache local.

## <a name="syntax"></a>Sintaxe

```
bitsadmin /Cache /Clear 
```

## <a name="BKMK_examples"></a>Exemplos

O exemplo a seguir limpa o cache local.
```
C:\>bitsadmin /Cache /Clear 
```
