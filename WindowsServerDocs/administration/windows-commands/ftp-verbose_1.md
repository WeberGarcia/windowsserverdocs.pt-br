# <a name="ftp-verbose_1"></a>FTP: verbose_1

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Alterna o modo **detalhado** .   
## <a name="syntax"></a>Sintaxe  
```  
verbose  
```  
### <a name="parameters"></a>Parâmetros  
nenhuma  
## <a name="remarks"></a>Comentários  
Por padrão, o **Verbose** está ativado.  Quando **Verbose** está ativado, todas as respostas de **FTP** são exibidas.  Quando uma transferência de arquivo é concluída, as estatísticas referentes à eficiência da transferência também são exibidas.  
## <a name="BKMK_Examples"></a>Disso  
Ativar e desativar o modo **detalhado** .  
```  
verbose  
```
