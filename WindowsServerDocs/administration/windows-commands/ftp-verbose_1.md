# <a name="ftp-verbose1"></a>FTP: verbose_1

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Alterna **detalhado** modo.   
## <a name="syntax"></a>Sintaxe  
```  
verbose  
```  
### <a name="parameters"></a>Parâmetros  
nenhuma  
## <a name="remarks"></a>Comentários  
Por padrão, **detalhado** está em.  Quando **verbose** está ativada, todos os **ftp** respostas serão exibidas.  Quando uma transferência de arquivos for concluída, as estatísticas sobre a eficiência da transferência também são exibidas.  
## <a name="BKMK_Examples"></a>Exemplos  
Ativar/desativar **detalhado** modo intermitente.  
```  
verbose  
```
