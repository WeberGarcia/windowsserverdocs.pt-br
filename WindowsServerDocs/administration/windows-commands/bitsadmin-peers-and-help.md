# <a name="bitsadmin-peers-and-help"></a>Bitsadmin colegas e ajuda

Exibe o uso da linha de comando para os comutadores **/Peers** .

## <a name="syntax"></a>Sintaxe

```
bitsadmin /peers /help
```

## <a name="examples"></a><a name=BKMK_examples></a>Disso

O comando a seguir exibe a ajuda de linha de comando para os comutadores **/Peers** .

```
C:\>bitsadmin /peers /help
```
