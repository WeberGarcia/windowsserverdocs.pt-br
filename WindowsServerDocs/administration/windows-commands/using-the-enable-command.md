# <a name="using-the-enable-command"></a>Usando o comando Enable

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Permite que todos os serviços para os serviços de implantação do Windows.
## <a name="subcommands"></a>Subcomandos
|Subommand|Descrição|
|-------|--------|
|[Usando o comando enable-servidor](using-the-enable-server-command.md)|Permite que todos os serviços em um servidor de serviços de implantação do Windows especificado (servidor de implantação).|
|[Usando o comando enable-TransportServer](using-the-enable-transportserver-command.md)|Permite que todos os serviços em um servidor de transporte especificado.|
