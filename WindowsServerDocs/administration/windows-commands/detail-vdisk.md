# <a name="detail-vdisk"></a>Detalhar vdisk

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Exibe as propriedades do disco rígido virtual selecionado \(VHD\).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
detail vdisk  
```  
  
## <a name="remarks"></a>Comentários  
  
-   Um VHD deve ser selecionado para essa operação seja bem-sucedida. Use o **selecionar o vdisk** comando para selecionar um vdisk e mudar o foco a ele.  
  
## <a name="BKMK_examples"></a>Exemplos  
Para ver detalhes sobre o VHD selecionado, digite:  
  
```  
detail vdisk  
```
