# <a name="hostname"></a>hostname

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Exibe a parte do nome de host do nome completo do computador do computador. 
## <a name="syntax"></a>Sintaxe
```
hostname
```
### <a name="parameters"></a>Parâmetros
|Parâmetro|Descrição|
|-------|--------|
|/?|Exibe a ajuda no prompt de comando.|
## <a name="remarks"></a>Comentários
Esse comando está disponível somente se o protocolo de Internet Protocol (TCP/IP) é instalado como um componente nas propriedades de um adaptador de rede na rede.
## <a name="BKMK_Examples"></a>Exemplos
Para exibir o nome do computador, digite:
```
hostname
```
