# <a name="bitsadmin-help"></a>bitsadmin help



Exibe o uso de linha de comando.

## <a name="syntax"></a>Sintaxe

```
bitsadmin /help | /?
```

## <a name="BKMK_examples"></a>Exemplos

O exemplo a seguir recupera a Ajuda de linha de comando.
```
C:\>bitsadmin /help
```
