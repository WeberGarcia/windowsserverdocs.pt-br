# <a name="bitsadmin-peers-and-clear"></a>pares de Bitsadmin e clear



Limpa a lista de ponto a ponto.

## <a name="syntax"></a>Sintaxe

```
bitsadmin /Peers /Clear 
```

## <a name="BKMK_examples"></a>Exemplos

O exemplo a seguir limpa a lista de ponto a ponto.
```
C:\>bitsadmin /Peers /Clear
```
