# <a name="bitsadmin-peers-and-discover"></a>Bitsadmin pares e descobrir



Descobre pares novamente.

## <a name="syntax"></a>Sintaxe

```
bitsadmin /Peers /Discover 
```

## <a name="BKMK_examples"></a>Exemplos

O exemplo a seguir detecta pares.
```
C:\>bitsadmin /Peers /Discover
```
