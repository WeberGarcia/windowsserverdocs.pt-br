# <a name="detail"></a>Detalhes



Exibe informações sobre o disco selecionado, a partição, o volume ou o disco rígido virtual (VHD).

## <a name="syntax"></a>Sintaxe

```
detail disk
detail partition
detail volume 
detail vdisk
```

## <a name="parameters"></a>Parâmetros

|Parâmetro|Descrição|
|---------|-----------|
|[disco de detalhe](detail-disk.md)|Exibe as propriedades do disco selecionado e os volumes existentes nele.|
|[partição de detalhe](detail-partition.md)|Exibe as propriedades da partição selecionada.|
|[volume de detalhe](detail-volume.md)|Exibe os discos no qual reside o volume atual.|
|[Detalhar vdisk](detail-vdisk.md)|Exibe as propriedades do VHD selecionada.|
