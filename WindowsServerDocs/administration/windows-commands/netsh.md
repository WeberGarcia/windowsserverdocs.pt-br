# <a name="netsh"></a>netsh



O **netsh** é um utilitário de script de linha de comando que permite que você, local ou remotamente, exiba ou modifique a configuração de rede de um computador atualmente em execução.

Há diferenças funcionais entre os comandos **netsh** no windows server® 2003, no windows server® 2008 e no windows server® 2008 R2:
-   Para obter mais informações sobre comandos **netsh** no Windows Server 2003, consulte [netsh](https://technet.microsoft.com/library/cc779693(v=ws.10).aspx).
-   Para obter mais informações sobre comandos **netsh** para o windows Server 2008 e o windows Server 2008 R2, consulte [referência técnica do netsh](https://technet.microsoft.com/library/cc754753(v=ws.10).aspx).
