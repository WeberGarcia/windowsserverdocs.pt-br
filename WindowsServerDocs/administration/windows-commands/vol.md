# <a name="vol"></a>vol



Exibe o rótulo de volume e o número de série do disco, se existirem.  Se usado sem parâmetros, **Vol** exibe informações para a unidade atual.

## <a name="syntax"></a>Sintaxe

```
vol [<Drive>:]
```

### <a name="parameters"></a>Parâmetros

|Parâmetro|Descrição|
|---------|-----------|
|> da unidade de \<:|Especifica a unidade que contém o disco para o qual você deseja exibir o rótulo de volume e o número de série.|
|/?|Exibe a ajuda no prompt de comando.|
