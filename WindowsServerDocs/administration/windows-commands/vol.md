# <a name="vol"></a>vol



Exibe o rótulo do volume de disco e o número de série, se existirem.  Se usado sem parâmetros, **vol** exibe informações para a unidade atual.

## <a name="syntax"></a>Sintaxe

```
vol [<Drive>:]
```

## <a name="parameters"></a>Parâmetros

|Parâmetro|Descrição|
|---------|-----------|
|\<Drive>:|Especifica a unidade que contém o disco para o qual você deseja exibir o número de série e o rótulo do volume.|
|/?|Exibe a ajuda no prompt de comando.|
