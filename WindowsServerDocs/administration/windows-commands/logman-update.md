# <a name="logman-update"></a>logman update



Atualizar um coletor de dados existente.

## <a name="syntax"></a>Sintaxe

```
logman update <counter | trace | alert | cfg | api> <[-n] <name>> [options]
```

### <a name="parameters"></a>Parâmetros

|Parâmetro|Descrição|
|---------|-----------|
|[contador de atualização do logman](logman-update-counter.md)|Atualizar um coletor de dados de contador.|
|[rastreamento de atualização do logman](logman-update-trace.md)|Atualizar um coletor de dados de rastreamento.|
|[alerta de atualização de logman](logman-update-alert.md)|Atualizar um coletor de dados de alerta.|
|[cfg de atualização de logman](logman-update-cfg.md)|Atualizar um coletor de dados de configuração.|
|[API de atualização do logman](logman-update-api.md)|Atualizar um coletor de dados de rastreamento de API.|
