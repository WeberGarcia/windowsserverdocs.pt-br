# <a name="pbadmin"></a>pbadmin

> Aplica-se a: Windows Server 2003, Windows Server 2003 R2, Windows Server 2003 com SP1, Windows Server 2003 com SP2

>[!IMPORTANT]
> Este comando foi preterido.

Administra os catálogos telefônicos. Usado sem parâmetros, o pbadmin inicia o administrador do catálogo telefônico. Para obter descrições e informações de uso, consulte [pbadmin](https://docs.microsoft.com/previous-versions/orphan-topics/ws.10/cc755767(v=ws.10)).
