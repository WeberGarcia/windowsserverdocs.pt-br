# <a name="vssadmin"></a>Vssadmin

>Aplica-se a: Windows 10, Windows 8.1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012, Windows Server 2008 R2, Windows Server 2008

Exibe os backups de cópia de sombra de volume atual e todos os gravadores de cópia de sombra instalados e provedores. Selecione um nome de comando no modo de exibição de tabela a seguir em sua sintaxe de comando.

|Comando|Descrição|
|---|---|
|[Vssadmin adicionar shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788051(v%3dws.11))|Adiciona uma associação de armazenamento de cópias de sombra de volume.|
|[Vssadmin criar sombra](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788055(v%3dws.11))|Cria uma nova cópia de sombra de volume.|
|[Sombras de exclusão vssadmin](vssadmin-delete-shadows.md)|Exclui as cópias de sombra de volume.|
|[Vssadmin delete shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc785461(v%3dws.11))|Exclui as associações de armazenamento de cópia de sombra de volume.|
|[Provedores de lista vssadmin](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788108(v%3dws.11))|Listas registrado provedores de cópia de sombra de volume.|
|[Vssadmin lista sombras](vssadmin-list-shadows.md)|Lista os existentes cópias de sombra de volume.|
|[Vssadmin lista shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788045(v%3dws.11))|Lista todas as associações de armazenamento de cópia de sombra no sistema.|
|[Volumes de lista vssadmin](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788064(v%3dws.11))|Lista de volumes que estão qualificados para cópias de sombra.|
|[Gravadores de lista vssadmin](vssadmin-list-writers.md)|Lista todos os gravadores de cópias de sombra de volume inscritos no sistema.|
|[Vssadmin shadowstorage de redimensionamento](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788050(v%3dws.11))|Redimensiona o tamanho máximo de uma associação de armazenamento de cópia de sombra.|
