# <a name="vssadmin"></a>Vssadmin

>Aplica-se a: 10 do Windows, Windows 8.1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012, Windows Server 2008 R2, Windows Server 2008

Exibe os backups de cópia de sombra de volume atual e todos os provedores e escritores de cópia de sombra instalados. Selecione um nome de comando na exibição da tabela a seguir sua sintaxe de comando.

|Comando|Descrição|
|---|---|
|[Vssadmin adicionar shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788051(v%3dws.11))|Adiciona uma associação de armazenamento de cópia de sombra de volume.|
|[Vssadmin criar sombra](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788055(v%3dws.11))|Cria uma nova cópia de sombra de volume.|
|[Vssadmin delete sombras](vssadmin-delete-shadows.md)|Exclui as cópias de sombra de volume.|
|[Vssadmin delete shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc785461(v%3dws.11))|Exclui associações de armazenamento de cópia de sombra de volume.|
|[Provedores de lista de vssadmin](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788108(v%3dws.11))|Provedores de cópia de sombra de volume registrados de listas.|
|[Vssadmin sombras de lista](vssadmin-list-shadows.md)|Listas existentes de cópias de sombra de volume.|
|[Vssadmin lista shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788045(v%3dws.11))|Lista todas as associações de armazenamento de cópia de sombra no sistema.|
|[Vssadmin volumes de lista](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788064(v%3dws.11))|Lista os volumes que estão qualificados para cópias de sombra.|
|[Vssadmin escritores de lista](vssadmin-list-writers.md)|Lista todos os autores de cópia de sombra de volume inscritos no sistema.|
|[Vssadmin redimensionamento shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788050(v%3dws.11))|Redimensiona o tamanho máximo de uma associação de armazenamento de cópia de sombra.|
