# <a name="ftp-trace_1"></a>FTP: trace_1



Alterna o rastreamento de pacotes.

## <a name="syntax"></a>Sintaxe

```
trace
```

#### <a name="parameters"></a>Parâmetros

none

## <a name="remarks"></a>Comentários

**Trace** exibe a série de chamadas de função FTP internas ao executar um comando **FTP** .

## <a name="examples"></a><a name=BKMK_Examples></a>Disso

Ativar e desativar o rastreamento.
```
trace
```
