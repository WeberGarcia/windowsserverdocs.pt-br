# <a name="wbadmin-get-disks"></a>Wbadmin get discos



Lista os discos internos e externos que estão atualmente online para o computador local.

Para listar os discos que estão online com este subcomando, você deve ser um membro do **operadores de Backup** grupo ou o **administradores** grupo, ou você deve ter sido recebido as permissões apropriadas. Além disso, você deve executar **wbadmin** em um prompt de comando elevado. (Para abrir um atalho de prompt de comando com privilégios elevados **Prompt de comando**e, em seguida, clique em **executar como administrador**.)

## <a name="syntax"></a>Sintaxe

```
wbadmin get disks
```

## <a name="parameters"></a>Parâmetros

Esse subcomando não tem parâmetros.
