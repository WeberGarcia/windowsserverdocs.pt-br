# <a name="rcp"></a>rcp



Copia os arquivos entre computadores. Esse comando foi preterido. Você pode instalar o subsistema para aplicativos baseados em UNIX usando o Assistente para adicionar recursos. Para obter mais informações, consulte [componentes de interoperabilidade do UNIX do Windows Server 2008](https://go.microsoft.com/fwlink/?LinkId=191835) no site da Microsoft. Após a instalação, você pode, em seguida, abra uma janela de comando do Shell C ou o Korn Shell e executar **rcp**. Para obter mais informações, digite **man rcp** no prompt de Shell C ou o Korn Shell.
