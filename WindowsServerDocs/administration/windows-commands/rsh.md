# <a name="rsh"></a>rsh



Executa comandos em computadores remotos que executam o serviço ou daemon RSH. Esse comando foi preterido. Você pode instalar o subsistema para aplicativos baseados em UNIX usando o Assistente para adicionar recursos. Para obter mais informações, consulte [componentes de interoperabilidade do UNIX do Windows Server 2008](https://go.microsoft.com/fwlink/?LinkId=191835) no site da Microsoft. Após a instalação, você pode, em seguida, abra uma janela de comando do Shell C ou o Korn Shell e executar **rsh**. Para obter mais informações, digite **man rsh** no prompt de Shell C ou o Korn Shell.
