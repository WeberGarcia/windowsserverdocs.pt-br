# <a name="the-stop-server-command"></a>O comando de parada do servidor

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Para serviços para os serviços de implantação do Windows.
## <a name="subcommands"></a>Subcomandos
|Subcomando|Descrição|
|-------|--------|
|[Subcommand: stop-Server](subcommand-stop-server.md)|Interrompe todos os serviços em um servidor de serviços de implantação do Windows (servidor de implantação).|
|[Subcommand: stop-TransportServer](subcommand-stop-transportserver.md)|Interrompe todos os serviços em um servidor de transporte.|
