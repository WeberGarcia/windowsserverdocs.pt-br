# <a name="stop-server"></a>parar-servidor
> Aplica-se a: Windows Server (canal semestral), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Interrompe os serviços de serviços de implantação do Windows.

## <a name="subcommands"></a>Subcomandos
|Subcomando|Descrição|
|-------|--------|
|[Subcomando: Stop-Server](subcommand-stop-server.md)|Interrompe todos os serviços em um servidor de serviços de implantação do Windows (servidor de implantação).|
|[Subcomando: Stop-TransportServer](subcommand-stop-transportserver.md)|Interrompe todos os serviços em um servidor de transporte.|
