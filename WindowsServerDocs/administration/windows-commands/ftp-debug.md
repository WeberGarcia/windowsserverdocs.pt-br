# <a name="ftpdebug"></a>ftp:debug

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Alterna o modo de depuração.   
## <a name="syntax"></a>Sintaxe  
```  
debug  
```  
### <a name="parameters"></a>Parâmetros  
nenhuma  
## <a name="remarks"></a>Comentários  
-   Por padrão, a depuração está desativado.  
-   Quando a depuração está ativada, cada comando enviado ao computador remoto é exibido, precedido pelo caractere maior que (>).  
## <a name="BKMK_Examples"></a>Exemplos  
Ativar/desativar modo de depuração e desativar.  
```  
debug  
```
