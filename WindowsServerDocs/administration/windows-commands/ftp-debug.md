# <a name="ftpdebug"></a>FTP: Depurar

> Aplica-se a: Windows Server (canal semestral), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Alterna o modo de depuração.   
## <a name="syntax"></a>Sintaxe  
```  
debug  
```  
#### <a name="parameters"></a>Parâmetros  
none  
## <a name="remarks"></a>Comentários  
- Por padrão, a depuração está desativada.  
- Quando a depuração está ativada, cada comando enviado ao computador remoto é exibido, precedido pelo caractere maior que (>).  
  ## <a name="examples"></a>Exemplos  
  Ativar e desativar o modo de depuração.  
  ```  
  debug  
  ```
