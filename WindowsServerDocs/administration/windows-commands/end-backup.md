# <a name="end-backup"></a>Backup do final




Encerra uma sessão de backup completo e problemas de um **BackupComplete** evento com o estado do gravador apropriado, se necessário.

## <a name="syntax"></a>Sintaxe

```
end backup
```

## <a name="remarks"></a>Comentários

-   Esse comando deve ser usado para todos os backups completos.
