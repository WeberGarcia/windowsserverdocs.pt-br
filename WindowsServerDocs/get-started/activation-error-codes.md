# <a name="troubleshooting-activation-error-codes"></a>Solução de Problemas de Códigos de Erro de Ativação

**Usuários domésticos** Este artigo é destinado para uso por agentes de suporte e profissionais de TI. Caso esteja buscando mais informações sobre mensagens de erro de ativação do Windows, vá para os seguintes sites da Microsoft:

[Obter ajuda sobre erros de ativação do Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

Ao tentar usar uma MAK (chave de ativação múltipla) ou o KMS (Serviço de Gerenciamento de Chaves) para realizar a ativação de volume em um ou mais computadores baseados em Windows, você recebe mensagens de erro contendo códigos de erro específicos. Este artigo discute como solucionar esses erros.
