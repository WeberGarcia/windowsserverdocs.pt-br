# <a name="troubleshooting-activation-error-codes"></a>Solução de problemas de códigos de erro de ativação

**Usuários domésticos** neste artigo é destinado ao uso por agentes de suporte e os profissionais de TI. Se você estiver procurando para obter mais informações sobre mensagens de erro de ativação do Windows, vá para os seguintes sites da Microsoft:

[Obtenha ajuda com erros de ativação do Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

Quando você tentar usar uma chave de ativação múltipla (MAK) ou o serviço de gerenciamento de chaves (KMS) para realizar a ativação de Volume em um ou mais computadores baseados em Windows, mensagens de erro que contêm códigos de erro específicos. Este artigo discute como solucionar esses erros.
