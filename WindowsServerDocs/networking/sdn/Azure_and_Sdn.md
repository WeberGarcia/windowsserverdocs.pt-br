# <a name="microsoft-azure-and-software-defined-networking"></a>Microsoft Azure e Software definidos rede

>Aplica-se a: Windows Server (anual por canal), Windows Server 2016

Microsoft Azure é a plataforma de nuvem da Microsoft: uma coleção de cada vez maior de serviços integrados - computação, armazenamento, dados, redes e aplicativo - que ajudam você mova mais rápido, fazer mais e economizar dinheiro.  
  
Abordagem da Microsoft para redes de software definido inclui projetar, desenvolver e operam redes datacenter escala global para serviços como o Microsoft Azure. Os datacenters globais do Microsoft Azure realizar dezenas de milhares de alterações na rede todos os dias, o que é possível por causa de rede de software definido.  
  
Microsoft Azure é executado no mesmo servidor do Windows e plataforma Hyper-V que estão incluídos no Windows Server. Windows Server e do System Center incluem melhorias e as práticas recomendadas da experiência da Microsoft em operacional redes de datacenter de escala global, como o Microsoft Azure para você, para que você pode implantar as mesmas tecnologias de controle, automação e flexibilidade quando usando software projetado tecnologias de rede.  
  
Para obter mais informações, consulte [o que é Microsoft Azure? ](https://azure.microsoft.com/overview/what-is-azure/?WT.mc_id=azurebg_us_sem_bing_br_nontest_whatisazure_whatisazure&WT.srch=1).
