# <a name="choosing-a-branchcache-design"></a>Escolher um Design BranchCache

>Aplica-se a: Windows Server (anual por canal), Windows Server 2016

Você pode usar este tópico para saber mais sobre modos de BranchCache e selecionar os modos de melhor para a implantação.  
  
Você pode usar este guia para implantar BranchCache nos seguintes modos e combinações de modo.  
  
-   Todas as filiais são configuradas para o modo de cache distribuído.  
  
-   Todas as filiais são configuradas para o modo de cache hospedado e têm um servidor de cache hospedado no site.  
  
-   Algumas filiais são configurados para o modo de cache distribuído e alguns filiais têm um servidor de cache hospedado no site e são configurados para o modo de cache hospedado.  
  
A ilustração a seguir ilustra uma instalação de modo dual, com uma filial configurado para o modo de cache distribuído e uma filial configurado para o modo de cache hospedado.  
  
![Escolher um design BranchCache](../../media/Choosing-a-BranchCache-Design/bc_new_modes.jpg)  
  
Antes de implantar BranchCache, selecione o modo de que sua preferência para cada filial em sua organização.
