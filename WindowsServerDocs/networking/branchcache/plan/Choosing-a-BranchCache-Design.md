# <a name="choosing-a-branchcache-design"></a>Escolher um design para o BranchCache

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016

Você pode usar este tópico para saber mais sobre os modos do BranchCache e selecionar os modos de melhor para sua implantação.  
  
Você pode usar este guia para implantar o BranchCache nos seguintes modos e combinações de modo.  
  
-   Todas as filiais são configuradas para modo de cache distribuído.  
  
-   Todas as filiais estão configuradas para modo de cache hospedado e tem um servidor de cache hospedado no site.  
  
-   Algumas filiais são configuradas para modo de cache distribuído e algumas filiais tem um servidor de cache hospedado no site e são configuradas para modo de cache hospedado.  
  
A ilustração a seguir ilustra uma instalação de modo duplo, com um branch configurado para o modo de cache distribuído e uma filial configurado para o modo de cache hospedado.  
  
![Escolher um design do BranchCache](../../media/Choosing-a-BranchCache-Design/bc_new_modes.jpg)  
  
Antes de implantar o BranchCache, selecione o modo que preferir para cada filial em sua organização.
