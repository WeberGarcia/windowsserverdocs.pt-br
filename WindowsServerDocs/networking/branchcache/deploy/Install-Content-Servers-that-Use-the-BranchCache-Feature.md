# <a name="install-content-servers-that-use-the-branchcache-feature"></a>Instalar o conteúdo servidores que usam o recurso BranchCache

>Aplica-se a: Windows Server (anual por canal), Windows Server 2016

Para implantar os servidores de conteúdo são servidores Web Secure Hypertext Transfer Protocol (HTTPS), servidores Web Hypertext Transfer Protocol (HTTP) e o serviço de transferência inteligente em segundo plano (BITS)-servidores de sistema de site do aplicativo baseado em servidores, por exemplo, distribuição de ramificação do Windows Server Update Services (WSUS) e o System Center Configuration Manager, você deve instalar o recurso BranchCache, iniciar o serviço BranchCache e (para servidores WSUS somente) executar etapas adicionais de configuração.  
  
Veja os tópicos a seguir para implantar servidores de conteúdo.  
  
-   [Instalar o recurso BranchCache](Install-the-BranchCache-Feature.md)  
  
-   [Configurar o Windows Server Update Services e 40; WSUS e 41; Servidores de conteúdo](configure-wsus-content-servers.md)
