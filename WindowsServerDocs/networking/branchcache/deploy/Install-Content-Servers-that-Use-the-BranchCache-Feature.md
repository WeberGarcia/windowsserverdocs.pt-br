# <a name="install-content-servers-that-use-the-branchcache-feature"></a>Instalar servidores de conteúdo que usam o recurso BranchCache

>Aplicável a: Windows Server (canal semestral), Windows Server 2016

Para implantar servidores de conteúdo que são servidores Web do protocolo HTTPS (Hypertext Transfer Protocol), servidores Web HTTP e servidores de aplicativos baseados em BITS (serviço de transferência inteligente em segundo plano), como Windows Server Update Services ( WSUS) e Microsoft Endpoint Configuration Manager servidores do sistema de sites de distribuição de ramificações, você deve instalar o recurso do BranchCache, iniciar o serviço do BranchCache e (somente para servidores WSUS) executar etapas de configuração adicionais.  
  
Consulte os tópicos a seguir para implantar servidores de conteúdo.  
  
-   [Instalar o recurso BranchCache](Install-the-BranchCache-Feature.md)  
  
-   [Configurar Windows Server Update Services &#40;servidores&#41; de conteúdo do WSUS](configure-wsus-content-servers.md)
