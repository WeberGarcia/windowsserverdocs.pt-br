# <a name="install-content-servers-that-use-the-branchcache-feature"></a>Instalar servidores de conteúdo que usam o recurso BranchCache

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016

Para implantar servidores de conteúdo que são servidores Web Secure o protocolo (HTTPS), servidores Web de protocolo HTTP (Hypertext Transfer) e serviço de transferência inteligente em segundo plano (BITS)-com base em servidores de aplicativos, como o Windows Server Update Services ( O WSUS) e servidores de sistema do site de distribuição de ramificação do System Center Configuration Manager, você deve instalar o recurso BranchCache, iniciar o serviço BranchCache e (para servidores do WSUS somente), executar etapas de configuração adicionais.  
  
Consulte os tópicos a seguir para implantar servidores de conteúdo.  
  
-   [Instalar o recurso BranchCache](Install-the-BranchCache-Feature.md)  
  
-   [Configurar o Windows Server Update Services &#40;WSUS&#41; servidores de conteúdo](configure-wsus-content-servers.md)
