# <a name="install-and-configure-content-servers"></a>Instalar e configurar servidores de conteúdo

>Aplica-se a: Windows Server (anual por canal), Windows Server 2016

Quando você implanta BranchCache no modo de cache distribuído ou cache hospedado, você deve implantar um ou mais servidores de conteúdo em seu escritório principal ou na nuvem. Servidores de conteúdo que são servidores Web ou servidores de aplicativos usam o recurso BranchCache. Servidores de conteúdo são servidores de arquivos usam o BranchCache para o serviço de função de arquivos de rede da função de servidor Serviços de arquivo no Windows Server 2016.  
  
Veja os tópicos a seguir para implantar servidores de conteúdo.  
  
-   [Instalar o conteúdo servidores que usam o recurso BranchCache](../../branchcache/deploy/Install-Content-Servers-that-Use-the-BranchCache-Feature.md)  
  
-   [Instalar servidores de conteúdo de serviços de arquivo](../../branchcache/deploy/Install-File-Services-Content-Servers.md)
