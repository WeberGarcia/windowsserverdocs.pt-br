# <a name="install-and-configure-content-servers"></a>Instalar e configurar servidores de conteúdo

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016

Quando você implanta o BranchCache no modo de cache distribuído ou no modo de cache hospedado, você deve implantar um ou mais servidores de conteúdo no seu escritório principal ou na nuvem. Os servidores de conteúdo que são servidores Web ou servidores de aplicativos usam o recurso BranchCache. Servidores de conteúdo que são servidores de arquivos usar o BranchCache para o serviço de função de arquivos de rede da função de servidor Serviços de arquivo no Windows Server 2016.  
  
Consulte os tópicos a seguir para implantar servidores de conteúdo.  
  
-   [Instalar servidores de conteúdo que usam o recurso BranchCache](../../branchcache/deploy/Install-Content-Servers-that-Use-the-BranchCache-Feature.md)  
  
-   [Instalar servidores de conteúdo de serviços de arquivo](../../branchcache/deploy/Install-File-Services-Content-Servers.md)
