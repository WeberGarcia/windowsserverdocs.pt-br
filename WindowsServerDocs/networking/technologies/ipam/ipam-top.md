# <a name="ip-address-management-ipam"></a>IPAM (Gerenciamento de Endereços IP)

>Aplica-se a: Windows Server (canal semestral), Windows Server 2016

Gerenciamento de endereço IP (IPAM) é um conjunto integrado de ferramentas que permitem ponta a ponta planejamento, implantação, gerenciamento e monitoramento da infraestrutura de endereço IP, com uma experiência de usuário. O IPAM descobre automaticamente os servidores de infraestrutura de endereços IP e servidores DNS (Sistema de Nomes de Domínio) na sua rede e permite que você os gerencie com base em uma interface central.
