# <a name="ip-address-management-ipam"></a>Gerenciamento de endereço IP (IPAM)

>Aplica-se a: Windows Server (anual por canal), Windows Server 2016

Gerenciamento de endereço IP (IPAM) é um conjunto de ferramentas para habilitar a ponta a ponta planejar, implantar, gerenciar e monitoramento de sua infraestrutura de endereço IP, com uma experiência de usuário integrado. IPAM automaticamente detecta servidores de infraestrutura de endereço IP e servidores de sistema de nome de domínio (DNS) em sua rede e permite que você gerenciá-los a partir de uma interface central.
