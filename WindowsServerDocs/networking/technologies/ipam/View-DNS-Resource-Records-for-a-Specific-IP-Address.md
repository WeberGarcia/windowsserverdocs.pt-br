# <a name="view-dns-resource-records-for-a-specific-ip-address"></a>Registros de recurso DNS de modo de exibição para um endereço IP específico

>Aplica-se a: Windows Server (anual por canal), Windows Server 2016

Você pode usar este tópico para exibir os registros de recurso do DNS que estão associados com o endereço IP que você escolher.  
  
A associação ao grupo **administradores**, ou equivalente, é o requisito mínimo para executar este procedimento.  
  
### <a name="to-view-resource-records-for-an-ip-address"></a>Para exibir os registros de recurso para um endereço IP  
  
1.  No Gerenciador do servidor, clique em **IPAM**. O console de cliente IPAM aparece.  
  
2.  No painel de navegação, em **espaço de endereço IP**, clique em **inventário de endereço IP**. No painel de navegação inferior, clique em **IPv4** ou **IPv6**. O inventário de endereço IP aparece na exibição de pesquisa do painel de exibição. Localize e selecione o endereço IP registros de recurso cujo DNS que você deseja exibir.  
  
    ![Inventário de endereço IP de modo de exibição](../../media/View-DNS-Resource-Records-for-a-Specific-IP-Address/ipam_IPInventory_01.jpg)  
  
3.  No painel de exibição **modo de exibição detalhes**, clique em **registros de recurso DNS**. Os registros de recurso que estão associados com o endereço IP selecionado são exibidos.  
  
    ![Exibir registros de recurso do DNS](../../media/View-DNS-Resource-Records-for-a-Specific-IP-Address/ipam_IPInventory_02.jpg)
