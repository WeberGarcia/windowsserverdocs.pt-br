# <a name="high-performance-networking-hpn"></a>Rede de alto desempenho (HPN)

Redes de alto desempenho (HPNs) desempenham um papel nos requisitos de processamento de dados em tempo real. Por exemplo, atividades, como a replicação do data center, recuperação de desastres de datacenter e distribuídos de alto desempenho computação exigem que os dados de alto volume de transferência e baixa latência de rede. HPNs com recursos de conexão dinâmica tornam os recursos de rede de alto desempenho mais acessíveis e gerenciáveis.
