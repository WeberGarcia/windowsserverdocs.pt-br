# <a name="shielded-vms-for-tenants---deploying-a-shielded-vm-by-using-virtual-machine-manager"></a>VMs blindadas para locatários – Implantando uma VM blindada usando o Virtual Machine Manager

Se você tiver acesso ao System Center 2016 - Virtual Machine Manager (VMM), você pode implantar uma VM blindada para o qual um modelo VM blindado já foi criado. 

Para implantar uma VM blindada no VMM, use as instruções em [provisionar uma nova VM blindada](https://technet.microsoft.com/system-center-docs/vmm/scenario/guarded-vms#provision-a-new-shielded-vm).
